> **종합 입시 상담 시스템**입니다.

## 🎯 3계층 시스템 설명

### Layer 1: 원본 파일 보관 (`_원본파일/`)

**역할**: 대학 공시 자료, 입시결과 집계 파일 보관

```
원본 파일의 예:
- 서울대 2024 정시 합격선.xlsx
- 고려대 2024 수시 결과집계.pdf
- 연세대 2024년 입시가이드.hwp
- 관악구교육청_11개교_진학현황.xlsx
```

**특징**: 
- PDF, Excel, HWP 등 원본 형식 보존
- 링크로 참조 (Obsidian에서 클릭하면 외부 프로그램으로 열림)
- 추적성 및 출처 명확

---

### Layer 2: 표준화 분석 노트 (`🎓_대학별_입시결과/` + `🏫_학교별_합격결과/`)

**역할**: 원본 파일에서 핵심 데이터만 추출 정리

```
예시 구조 (각 대학별 파일):

| 전형 | 2022 | 2023 | 2024 | 변화추세 |
|------|------|------|------|---------|
| 수시 | ?명  | ?명  | ?명  | ↑/→/↓ |
| 정시 | ?명  | ?명  | ?명  | ↑/→/↓ |

수시 커트라인 (2024)
- 일반: 내신 1.5
- 지역: 내신 2.0

정시 커트라인 (2024)
- 국어2+수학1+영어1: 402점
- 표준점수 환산: ???

→ 원본 파일 링크: [📊 원본 자료](_원본파일/...)
```

**특징**:
- Markdown 표와 텍스트로 빠른 검색 가능
- Obsidian 내에서 바로 읽을 수 있음
- 원본과의 1:1 대응 (추적 가능)

---

### Layer 3: 상담 분석 도구 (`💡_상담분석도구/`) ⭐ 가장 중요

**역할**: 학부모/학생의 성적을 입력받아 **즉시 답변** 제시

```
상담 시나리오:
학부모: "우리 아이 내신이 1.8인데, 어느 대학을 노려볼까요?"

답변:
→ [[📊_내신_대학_매핑표|매핑표]] 참조
→ "내신 1.8이면..."
   ✅ SKY는 어렵고
   ✅ 중상위권 (서강/성균/한양) 수시 도전 가능
   ✅ 정시로는 상위권 (중경외시) 목표
→ [[📝_원서배치_시뮬레이터|시뮬레이터]]로 구체적 배치 제시
→ [[💬_입시상담_시스템|상담 결과 보고서]] 작성
```

---

## 📊 각 파일의 역할

### `💡_상담분석도구/` 안의 4개 파일

**1. [[📊_내신_대학_매핑표|📊 내신 → 대학 매핑표]]**
```
"내신이 몇 등급이면 어느 대학을 갈 수 있나?"에 답변
- 내신 1.0~1.2: SKY 도전 가능
- 내신 1.3~1.5: SKY는 어렵고, 중상위 도전
- 내신 1.6~2.0: 상위권 목표
- ... etc
```

**2. [[📊_수능_대학_매핑표|📊 수능 → 대학 매핑표]]**
```
"수능 몇 등급이면 정시로 어디를 갈까?"에 답변
- 두 가지 입력 방식 지원:
  a) 등급 입력 (국어2, 수학1, 영어1 형식)
  b) 표준점수 입력 (변환 테이블 제공)
```

**3. [[📊_전형별_비교분석|📊 전형별 비교 분석]]**
```
"수시 중심인가, 정시 중심인가?"에 답변
- 성적 수준별로 추천 전형 제시
  예) 내신1.8 + 수능2등급 → "수시 70% : 정시 30%"
- 학종 vs 교과 vs 논술 비교
```

**4. [[📝_원서배치_시뮬레이터|📝 원서배치 시뮬레이터]]**
```
"6개 대학을 어디에 배치할까?"에 답변
- 상향 1개 / 적정 2개 / 하향 2개 / 안전 1개 선정
- 각 대학의 합격 가능성 % 제시
- 근거: 실제 입시결과 데이터 + 합격 생기부
```

## 🔗 다른 시스템과의 연결

### 상담 흐름도
```
💬_입시상담_시스템/📌_상담_흐름도.md
    ↓ 사용
입시결과_DB/💡_상담분석도구/
    ↓ 근거
입시결과_DB/🎓_대학별_입시결과/ (실제 데이터)
    ↓ 참고
합격생기부_샘플/ (합격한 학생들은 어떤 프로필인가?)
    ↓ 기록
학생_생기부_컨설팅_기록/[학생명]/ (우리 학생의 성과)
```

### 최종 결과물
```
상담 결과 보고서
= 내신/수능 분석
  + 지원 가능 대학 범위
  + 수시/정시 추천비율
  + 원서 6개 배치안
  + 합격 가능성 %
  + 준비 로드맵
```

## 📝 원본 파일 넣는 방법

### Step 1: 파일 저장
```
1. Excel/PDF/HWP 파일 다운로드 또는 수집
2. 입시결과_DB/_원본파일/ 해당 폴더에 저장
   예) _원본파일/수시입시결과/서울대_2024수시.xlsx

3. 파일명은 명확하게:
   ❌ 자료.xlsx
   ✅ 서울대_2024수시_합격선.xlsx
```

### Step 2: 분석 노트에 링크 추가
```
입시결과_DB/🎓_대학별_입시결과/SKY/📊_서울대_입시결과.md 에서:

[📊 원본 파일 다운로드](_원본파일/수시입시결과/서울대_2024수시.xlsx)
```

### Step 3: 핵심 데이터를 표로 정리
```
위 파일에서:

| 연도 | 수시모집 | 수시합격 | 경쟁률 | 커트라인 |
|------|---------|---------|--------|----------|
| 2024 | 500명   | 500명   | 15:1  | 내신1.2 |
| 2023 | 500명   | 500명   | 18:1  | 내신1.5 |
```

### Step 4: 상담 도구에 반영
```
💡_상담분석도구/📊_내신_대학_매핑표.md 에서:

내신1.2 → SKY 수시 도전 가능
(근거: 서울대 2024년 커트라인 1.2)
```

## 💡 사용 예시

### 상담 상황 1: "내신 1.8인데 어디를 갈 수 있나요?"

**우리의 상담:**
```
Step 1: [[📊_내신_대학_매핑표|내신 매핑표]] 확인
→ "내신 1.8은... 중상위권 수시 도전 가능,
   상위권은 어려운 수준입니다"

Step 2: [[💬_입시상담_시스템|상담 흐름도]] 따라 진행
→ "수능 성적이 어떻게 되는가?"
→ "수시 vs 정시 비중 결정"

Step 3: [[📝_원서배치_시뮬레이터|시뮬레이터]] 실행
→ "내신1.8 + 수능 추정 → 이렇게 배치하세요"

Step 4: [[📋_상담_결과보고서_템플릿|보고서]] 작성
→ 학부모에게 제시할 문서 완성
```

### 상담 상황 2: "수능이 2등급인데 정시는?"

**우리의 상담:**
```
Step 1: [[📊_수능_대학_매핑표|수능 매핑표]] 확인
→ "수능 2등급이면 SKY 이과 가능 범위입니다"
→ 표준점수/등급 변환 안내

Step 2: [[📊_전형별_비교분석|전형별 비교]] 확인
→ "정시 중심으로 진행 추천"

Step 3: [[📝_원서배치_시뮬레이터|배치 시뮬레이터]]
→ 구체적인 6개 대학 추천

Step 4: [[📊_관악구_11개교_합격현황|관악구 내 선례]] 참고
→ "우리 지역에서 비슷한 학생들은..."
```

## 🎯 시스템의 강점

✅ **빠른 답변**: 매핑표만 보면 1분 내 답변 가능
✅ **근거 제시**: "왜? 어떤 데이터 기반?"을 명확히 설명
✅ **합격 사례**: 합격생기부 샘플과 연계한 현실적 조언
✅ **장기 추적**: 학생 기록으로 시간 경과에 따른 변화 추적
✅ **신뢰도**: 실제 입시결과 데이터 기반 (변덕 없는 객관적 조언)
