## 후속 조치 권고

### 수동 다운로드 우선순위

1. **한양대** (https://go.hanyang.ac.kr/web/result/result.do?m_type=SUSI)
   - 게시글 "2024-2026 전형별 입시결과" 클릭 → 첨부파일 다운로드
   - 2024·2025·2026학년도 수시·정시 통합 파일 포함 가능

2. **서울대** (https://admission.snu.ac.kr/materials/stats/competition_rate)
   - 2025학년도 정시·수시, 2024학년도 정시·수시 게시글 클릭 → 첨부파일 다운로드

3. **대학어디가** (https://www.adiga.kr/uct/ces/archiveView.do?menuId=PCUCTCES1000&prtlBbsId=24798)
   - 모집학년도 "2025", "2024" 선택 → 지역별 전형결과 PDF 다운로드
   - 광주·제주 외 지역 자료도 존재 가능 (총 2,427건)

4. **고려대** (https://oku.korea.ac.kr/oku/cms/FR_CON/index.do?MENU_ID=600&SITE_NO=2)
   - 공지사항 섹션에서 입시결과 게시글 탐색
