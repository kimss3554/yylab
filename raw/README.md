# 원본 자료 (Raw)

이 폴더는 **불변(Immutable)** 원본 저장소입니다.

- 논문, 기사, 회의록, 슬랙보내기, 팟캐스트 메모, 데이터 파일 등
- LLM 에이전트는 **읽기만** 하고, 파일을 수정·삭제하지 않습니다
- 웹 클립(Obsidian Web Clipper 등)은 여기 또는 하위 폴더에 저장

## 이미지

`assets/`에 로컬로 저장하면 LLM이 URL 없이 참조할 수 있습니다.  
Obsidian: 설정 → 파일 및 링크 → 첨부 파일 폴더를 `raw/assets`로 지정하는 것을 권장합니다.
