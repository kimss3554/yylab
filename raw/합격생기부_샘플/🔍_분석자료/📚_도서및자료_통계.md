## 📰 논문 및 학술자료

```
분야별:
- 과학 논문: ??건 (고등학교 수준 논문)
- 수학 논문: ??건 (응용수학, 통계)
- 경제 논문: ??건

특징:
→ 학술지보다는 교양 수준 자료

**과학 계열 학생들**
```

- 탐구보고서에 인용

예시: [학생6]
"현재 의학 연구 동향을 파악하기 위해 
△△에 관한 최신 논문을 검토했다"
```

**인문 계열 학생들**
```
- 논문보다 단행본 중심
- 기사/신문 기사 활용

```
전체 평균: ?? 권 (고등학교 3년)

진로별:

- 경영: ?? 권 (경제/경영)

```

### 독서량과 합격의 상관관계

**발견:**
```
✅ 많은 책보다 "깊이있는 읽기"
   → 5권을 깊게 vs 20권을 얕게
   → 생기부에 "깊이" 기재 가능

✅ 진로와 연관된 책 우선순위
   → 랜덤한 독서보다 전략적 독서

✅ 책 읽기 → 활동으로의 연결
   → 단순 "읽었습니다"가 아닌
   → "이 책을 바탕으로 △△를 했습니다"
```

## 💡 독서 전략 가이드

### 1학년 때 해야 할 독서

**목표**: 기초 역량 형성

```
인문계:
- 고전 문학 2-3권
- 철학 입문서 1-2권
- 역사서 1-2권
- 총 5-6권

자연계:
- 과학 교양서 3-4권
- 수학 관련서 1-2권
- 총 5-6권
```

**전략:**
```
→ "교과와 연결된 책" 읽기
→ 읽은 책을 세특에 기재될 수 있도록
```

### 2학년 때 해야 할 독서

**목표**: 진로 심화

```
의학 지망:
- 의학 관련 도서 2-3권
- 과학 심화서 2-3권
- 윤리/철학 1권

공학 지망:
- 공학 관련 도서 2-3권
- 수학 심화서 2-3권
- 기술 트렌드 도서 1권

경영 지망:
- 경제학 기초 2-3권
- 경영학 도서 2-3권
- 리더십 도서 1권
```

**전략:**
```
→ 세특이 길어지는 시기 (고2)
→ 읽은 책이 세특에 다중 기재될 기회
→ 탐구보고서 주제와 연계
```

### 3학년 때 해야 할 독서

**목표**: 진로 확정 + 대입 준비

```
시간이 부족하므로:
- 1~2권의 진로 도서 심화
- 이전에 읽은 책 복습
- 신문/기사 활용
```
