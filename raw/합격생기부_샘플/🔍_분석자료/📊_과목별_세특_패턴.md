### 강조되는 역량

| 역량 | 언급도 | 표현 방식 |
|------|--------|----------|

| **창의성** | ⭐⭐⭐⭐ | "독창적인 해석", "창의적인 표현" |
| **표현력** | ⭐⭐⭐⭐ | "우수한 표현능력", "정확한 어휘 사용" |
| **독서능력** | ⭐⭐⭐⭐ | "다양한 도서 읽기", "깊이있는 이해" |
| **토론능력** | ⭐⭐⭐ | "활발한 참여", "설득력있는 주장" |

### 자주 나오는 문구

**Activity-기반**
```
"○○를 읽고..." → [학생3] SKY 합격
"△△에 대해 토론하며..." → [학생3] [학생5]
"세 차례의 수행평가에서..." → [학생3]
```

**역량-기반**
```
"비판적 사고를 통해..." → [학생3] [학생5] [학생9]
"창의적으로 해석..." → [학생1] [학생3]
"논리적으로 구성..." → [학생3] [학생5]
```

**평가-기반**
```
"우수한 수준으로 평가..." → [학생3] [학생6]
"탁월한 능력 보임..." → [학생3]
"뛰어난 역량 발휘..." → [학생1]
```

### 기재 구조 분석

**공통 구조**: [활동/내용] + [역량표현] + [구체적성과]

**예시 1: [학생3] - SKY 합격**
```
원문: "현대소설의 특징과 의미를 비판적으로 분석하고, 
      다양한 관점의 해석을 검토하는 과정에서 
      문학적 사고와 논리적 표현능력이 우수함을 보여주었다"

구조:
- 활동: 현대소설 분석
- 역량: 비판적분석, 논리적표현
- 성과: 우수함
```

**예시 2: [학생1]**
```
원문: "독서토론 수업에서 제시된 도서에 대해 
      여러 주제로 창의적이고 논리적인 의견을 제시하며, 
      다양한 해석과 관점을 존중하는 태도로 참여했다"

구조:
- 활동: 독서토론
- 역량: 창의성, 논리성, 포용성
- 성과: 활발한 참여
```

### 국어 세특 최강 전략

**합격 사례에서 배우는 것:**

✅ **구체적 도서명 언급**
```
좋은 예: "○○의 [도서명]을 읽고..."
나쁜 예: "여러 도서를 읽고..."
```

✅ **구체적 활동 기술**
```
좋은 예: "3차례의 수행평가에서 △△주제로..."
나쁜 예: "수행평가에서 좋은 결과"
```

✅ **역량 + 근거의 조합**
```
좋은 예: "비판적 사고를 바탕으로 △△와 ○○를 비교 분석했다"
나쁜 예: "비판적 사고능력이 뛰어나다"
```

## 📌 수학 세특 패턴

### 기재 특징

**기재율**: ??% (약 85%)
**평균 글자수**: ??자 (70~140자)
**특징**: 국어보다 평균적으로 짧은 경향

### 강조되는 역량

| 역량 | 언급도 | 표현 방식 |
|------|--------|----------|
| **문제해결력** | ⭐⭐⭐⭐⭐ | "적극적으로 문제 해결", "창의적 풀이" |
| **수학적사고** | ⭐⭐⭐⭐ | "수학적 직관력", "논리적 접근" |
| **탐구심** | ⭐⭐⭐⭐ | "심화학습", "깊이있는 이해" |
| **적극성** | ⭐⭐⭐ | "주도적으로", "능동적으로" |

### 자주 나오는 문구

**Activity-기반**
```
"탐구활동에서..." → [학생2] [학생8]
"고난도 문제를 해결하며..." → [학생2] [학생6]
"과제연구를 통해..." → [학생1]
```

**역량-기반**
```
"문제해결력이 뛰어나다" → [학생2] [학생6]
"수학적 직관이 있다" → [학생2]
"창의적인 풀이" → [학생1]
```

### 특이한 점

**문제**: "국어보다 글자수가 짧은 경향"
```
평균 국어: 120자
평균 수학: 95자

원인 추측:
- 객관적 평가가 가능해서 (등급으로 충분)
- 수학 용어 자체가 간결함
- 활동이 명확해서 설명이 간단함
```

**대응방안**: 우리 학생들에게
```
수학에서는 글자수보다 "구체성"
→ "○○ 주제의 심화 탐구를 통해..."
→ "△△를 이용한 창의적 문제풀이..."
```

### 수학 세특 최강 전략

✅ **구체적 탐구 주제**
```
좋은 예: "[주제명]: ○○ 함수의 응용"
나쁜 예: "탐구활동"
```

✅ **구체적 활동**
```
좋은 예: "미적분 심화 개념을 다중 관점에서 분석"
나쁜 예: "수학을 잘함"
```

✅ **수학 용어 활용**
```
좋은 예: "수렴과 발산의 개념을 실생활에 적용"
나쁜 예: "어려운 개념을 이해함"
```

**기재율**: ??% (약 75%)
**평균 글자수**: ??자 (70~120자)
**기재 가능성**: 국어, 수학보다 낮은 편

### 강조되는 역량

| 역량 | 언급도 | 표현 방식 |
|------|--------|----------|
| **의사소통능력** | ⭐⭐⭐⭐ | "능숙한 영어 표현", "정확한 발음" |
| **국제감각** | ⭐⭐⭐ | "국제적 이해", "다문화 감각" |
| **학습태도** | ⭐⭐⭐ | "적극적 참여", "높은 관심" |
| **창의성** | ⭐⭐ | "창의적 표현", "독창적 발상" |

### 자주 나오는 문구

**Activity-기반**
```
"영어 발표에서..." → [학생4] [학생10]
"원문 읽기를 통해..." → [학생1]
"영어로 작성한 보고서..." → [학생3]
```

### 특이한 점

**낮은 기재율의 이유**
```
1) 영어는 "등급"으로 충분
2) 특이한 활동이 적음
3) 다른 과목보다 정형화됨
```

**기재되는 학생들의 특징**
```
→ 영어 발표/토론에 참여
→ 원문 읽기 등 심화활동
→ 국제 관심사 표현
```

## 📌 과학 세특 패턴

### 기재 특징

**기재율**: ??% (약 90%)
**평균 글자수**: ??자 (100~180자)
**가장 길게 기재되는 과목** ← 중요!

### 강조되는 역량

| 역량 | 언급도 | 표현 방식 |
|------|--------|----------|
| **탐구력** | ⭐⭐⭐⭐⭐ | "심화 실험 설계", "자발적 탐구" |
| **과학적사고** | ⭐⭐⭐⭐ | "과학적 방법론", "논리적 해석" |
| **실험능력** | ⭐⭐⭐⭐ | "정밀한 실험수행", "과학적 설계" |
| **창의성** | ⭐⭐⭐ | "창의적 실험", "새로운 접근" |

### 자주 나오는 문구

**Activity-기반**
```
"실험실습에서..." → 모든 학생
"탐구보고서 작성..." → [학생1] [학생2] [학생14]
"심화 실험을 설계하며..." → [학생6]
```

**역량-기반**
```
"뛰어난 탐구력" → [학생1] [학생2] [학생6]
"과학적으로 접근" → [학생1]
"창의적인 실험 설계" → [학생2] [학생6]
```

### 과학이 특별한 이유

**✅ 긴 글자수**
```
국어/수학/영어: 70~150자
과학: 100~180자 (가장 김)

이유: 구체적 실험 설명 가능
```

**✅ 높은 기재율**
```
과학: 약 90%
다른 과목: 70~85%

이유: 성적과 별개로 기재 가능
```

**✅ 차등 표현 가능**
```
동일 성적(같은 등급)도 세특으로 구별됨
→ 탐구력이 있으면 기재됨
```

### 과학 세특 최강 전략

✅ **구체적 탐구 주제**
```
좋은 예: "[주제]: △△의 △△에 대한 탐구"
나쁜 예: "과학 탐구"
```

✅ **구체적 설계 과정**
```
좋은 예: "○○의 기울기를 달리하여 △△의 변화를 측정"
나쁜 예: "실험을 잘함"
```

✅ **과학적 해석**
```
좋은 예: "결과 분석을 통해 □□ 원리를 도출"
나쁜 예: "실험 결과가 좋음"
```

## 💡 학생 대상 가이드

### "내 과목별 세특을 어떻게 극대화할까?"

**국어를 공략하려면:**
```
1) 정확한 도서명 + 저자명
2) 구체적 활동 (몇 차시, 어떤 주제)
3) 역량 + 근거의 명확한 제시
4) 글자수: 120~150자 목표
```

**수학을 공략하려면:**
```
1) 구체적 탐구 주제명
2) 심화 개념의 자세한 설명
3) 문제 해결의 과정 강조
4) 글자수: 90~130자 목표
```

**영어를 공략하려면:**
```
1) 발표/토론/읽기 등 구체적 활동
2) 영어 능력 + 국제감각 병행
3) 원문 읽기 등 심화 활동
4) 글자수: 80~120자 목표
```

**과학을 공략하려면:**
```
1) 구체적 실험명 + 주제
2) 실험 설계 과정의 창의성
3) 과학적 해석과 결론
4) 글자수: 130~170자 목표 (길수록 좋음)
```
