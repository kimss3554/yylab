## 자료 출처

### 공식 채널
1. **학교알리미** (https://www.schoolinfo.go.kr)
   - 교육부 공식 학교정보공시 포털
   - 모든 공립학교의 필수 공시 자료
   - 통계 자료(학생수, 진로현황 등)

2. **각 학교 공식 웹사이트**
   - 학사일정, 특색사업 계획 등
   - 공지사항, 자료실 메뉴에서 조회

### 자료 신뢰성
- 모든 자료는 교육부 공시 기준 또는 학교 공식 문서
- 개인정보는 삭제된 공개용 버전

## 작업 가이드

### 새로운 자료 추가
1. 학교 폴더에 파일 저장
2. 파일명에 연도 포함 (예: `학사일정_2026.pdf`)
3. `다운로드_현황_2026-07-01.md` 에서 체크박스 업데이트

### 공시 일정 확인
- **1월~3월**: 신학년도 교육계획, 학사일정
- **3월~5월**: 학년별학생수 확정, 졸업생진로현황
- **8월~9월**: 2학기 방과후활동 등

### 문제 해결
**자료를 찾을 수 없음**
1. 학교알리미(schoolinfo.go.kr) 재확인
2. 학교 공식 웹사이트 검색
3. 교무실에 직접 문의
