# 인하대학교 논술 자료 URL 목록

## 입학처
- 메인: https://admission.inha.ac.kr
- SSO 로그인: https://idp.inha.ac.kr:8443/ (로그인 필요 시 리다이렉트)

## 수집 현황 (2026-06-14)
- 입학처 사이트: 일부 페이지 SSO 로그인 리다이렉트
- 논술 전형: 인하대 논술우수자 전형 실시
- 공개 자료 URL: 미확보 (로그인 없이 접근 가능 여부 미확인)

## 조치 필요
- 브라우저에서 https://admission.inha.ac.kr 접속
- 로그인 없이 논술>기출문제 페이지 접근 시도
- 공개 PDF URL 확보 또는 로그인 후 다운로드
