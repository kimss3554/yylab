## 학년군별 핵심 내용 체계 (2022 개정)

### 3~4학년군

| 영역 | 핵심 개념 | 성취기준 예시 |
|------|----------|-------------|
| 지리 | 고장·지역의 위치와 특성 | 고장의 위치, 지리적 특성, 생활 모습 비교 |
| 역사 | 우리 고장의 역사와 문화유산 | 지역 문화유산의 역사적 의미 탐구 |
| 일반사회 | 지역의 사회적 특성과 공공 기관 | 공공 기관의 역할, 지역 문제 해결 방법 |

### 5~6학년군

| 영역 | 핵심 개념 | 성취기준 예시 |
|------|----------|-------------|
| 지리 | 국토의 자연환경·인문환경 | 자연환경과 인간 생활의 관계 분석 |
| 역사 | 우리나라 역사 흐름 | 시대별 특징, 역사적 사건의 원인과 결과 |
| 일반사회 | 민주주의·경제·세계화 | 민주정치 원리, 경제활동의 특성, 세계 속 우리나라 |

## 문제 구성 원칙

1. **성취기준 연계**: 각 문제는 교육과정 성취기준 코드 표기
2. **인지 수준 분류**: 이해 → 적용 → 분석 → 평가
3. **채점 기준표 첨부**: 모든 문제에 예시 답안 + 채점 기준 포함
4. **교과서 중립**: 특정 출판사 교과서에 종속되지 않는 교육과정 기반 출제
