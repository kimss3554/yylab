# 서논술형 평가 문제집

**기반:** 2022 개정 교육과정 (교육부 고시 제2022-33호)

## 폴더 구조

```
workbooks/
├── elementary/          # 초등학교
│   ├── social/          # 사회 (3~6학년)
│   │   ├── grade3-4/    # 3~4학년군
│   │   └── grade5-6/    # 5~6학년군
│   ├── korean/          # 국어
│   ├── math/            # 수학
│   ├── science/         # 과학
│   └── moral/           # 도덕
└── middle/              # 중학교
    ├── social/          # 사회
    ├── history/         # 역사
    ├── korean/          # 국어
    ├── math/            # 수학
    ├── science/         # 과학
    ├── moral/           # 도덕
    └── english/         # 영어
```

## 문제 유형

| 유형 | 설명 | 배점 예시 |
|------|------|---------|
| 서술형 | 핵심 개념·사실을 2~4문장으로 서술 | 5~10점 |
| 논술형 | 논거를 갖춰 의견·분석을 논리적으로 서술 | 10~20점 |

## 파일 명명 규칙

`{학년}_{단원번호}_{문제번호}_{유형}.md`
예: `4학년_1단원_01_서술.md`

## 진행 현황

| 과목 | 학년 | 상태 |
|------|------|------|
| 초등 사회 3~4학년 | grade3-4 | 🔧 구축 중 |
| 초등 사회 5~6학년 | grade5-6 | ⏳ 대기 |
| 그 외 | - | ⏳ 교과서 자료 수집 후 착수 |
