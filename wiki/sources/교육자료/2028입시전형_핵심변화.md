# 2028 대입전형 핵심 변화

> **2028학년도 수험생** = 현재 고1 (2026년 3월 입학, 2022 개정 교육과정 첫 세대)

---

## 서울대 2028 주요 변화

| 항목 | 내용 |
|------|------|
| 시행계획 발표 | 2026년 4월 (잠정안, 대교협 심의 후 확정) |
| **정시 축소** | 정시 지역균형 폐지 → 수시 지역균형 확대 |
| **정시 일반전형만** | 정시는 일반전형 단일화 |
| 정시 1단계 | **3배수** 등급 선발 (3학년 2학기 성적 포함!) |
| 지역균형 | 고교당 추천 인원 3명 이내 / 자사고·특목고 지원 불가 |

### 서울대 2028 수시 지역균형 전형방법

- 1단계: 서류 100% (3배수)
- 2단계: 1단계 성적 + 면접
- 수능 최저: 확인 필요 (변경 예정)

---

## 고려대 2028 주요 변화

| 전형 | 2027학년도 | 2028학년도 |
|------|----------|----------|
| 학교추천(교과) | 교과 90% + 서류 10% | **교과 80% + 서류 20%** |
| 학업우수(종합) | 서류 100% 일괄 | **1단계 서류(5배수) → 2단계 면접 20%** |
| 계열적합(종합) | 1단계 서류 + 2단계 면접 40% | **서류 100% 일괄** |

**고려대 수능 최저 (2028):**
- 교과·학종 일반: 국·수·영·탐 3개 등급 합 7 이내 + 한국사 4등급
- 의학과: 4개 등급 합 5 이내

### 수시/정시 비율 (2028 고려대)
- 수시: 2,721명 (58.3%)
- 정시: 1,867명 (40.0%)

---

## 전반적인 2028 입시 트렌드

### 1. 정시 축소 / 수시 확대
서울대를 중심으로 정시 비중이 줄어드는 방향. 2025~2026학년도부터 적용된 정시 의무 비율(40%)이 완화될 것으로 예상.

### 2. 면접 강화
- 고려대 학업우수전형: 기존 서류 100% → 2028년에 면접 추가
- 서울대 지역균형: 면접 포함 유지
- 면접에서 교과 이수 내용·세특 기반 질문 증가

### 3. 교과 이수 평가 강화
- **서울대:** 교과이수 적절성, 교과 위계 (고급→심화 순서), 성취도+등급+원점수 종합 평가
- **고려대:** 학종에서 이수 과목의 전공 연계성 강조
- **경희대:** 2028학종 서류형 → 등급 평가 없이 **성취도만** 평가 (3등급/성취도A → 1등급 환산)

### 4. 권장·핵심 과목 공개
- 서울대·고려대·성균관대 등 주요 대학이 전공별 권장과목을 명시적으로 공개
- 이수 여부가 학종 평가에 반영됨
- **미이수자도 지원은 가능하나, 입사관이 교과 이수 적절성에서 감점 요인**

## 대학별 2028 핵심 권장과목 (자연계열 중심)

| 대학 | 주요 핵심 과목 | 권장 과목 |
|------|------------|---------|
| **서울대** | 수학1·수학2·미적분1, 물리학1·화학1 | 확통, 기하, 물리학Ⅱ(역학과에너지/전자기와양자) |
| **고려대** | 수학1·수학2·미적분1, 물리학1·화학1·화학2 | 확통, 기하, 물리학Ⅱ |
| **경희대** | (학종서류형은 성취도 평가 — 등급 미반영) | |

> **실전 포인트:** 고교에서 서울대·고려대 권장과목을 개설하지 않으면, 학생이 공동교육과정이나 온라인학교를 통해 이수해야 한다. 관악구에서 이 과목들을 자체 개설하는 학교가 어디인지가 컨설팅의 핵심.

## 수능 연계 (2028)

| 구분 | 수능 포함 과목 |
|------|-------------|
| 국어 | (1학년) 공통국어 내용 범위 |
| 수학 | 대수, 미적분Ⅰ, 확률과통계 |
| **수능 제외** | **미적분Ⅱ, 기하** (심화수학) |
| 과학 탐구 | 통합과학 내용 기반 |

> 심화수학(미적분Ⅱ·기하)은 수능 범위가 아니지만, 수시에서는 이수 여부가 평가됨. 수리논술에도 포함.

## 2028입시가 관악구 학교 선택에 주는 시사점

1. **권장과목 개설 여부 확인 필수** — 역학과에너지·전자기와양자·물질과에너지 등을 개설하는 학교와 아닌 학교의 차이가 실제 입시 결과로 이어짐
2. **미적분Ⅱ 개설 학교** — 관악구 내 남강고·문영여고만 개설. 이공계 최상위 지망 학생의 학교 선택 기준
3. **면접 준비** — 세특 기반 면접 강화에 따라 1학년부터 세특 전략적 관리 중요
