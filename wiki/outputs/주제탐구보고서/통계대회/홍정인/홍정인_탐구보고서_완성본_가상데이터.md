# 놀 곳이 없는 걸까, 우리에게 맞지 않는 걸까?
### — 전국 최초 청소년 전용공간을 갖춘 대치동, 왜 처음엔 외면받았고 무엇을 바꾸자 이용이 늘었나 —

## Ⅰ. 서론

### 1. 탐구 계기
나는 대치동에 산다. 대치동은 학원만 1,300~1,600개, 하루 10대 유동인구 3만~4만 명의 대한민국 사교육 중심지다. 시험이 끝나도 친구와 안전하게 쉴 곳을 찾으면 막상 갈 데가 없다고 느꼈다. 그런데 찾아보니 대치동에는 **전국 최초·유일의 청소년 전용공간**(사이'쉼', 스트레스 프리존 등)이 이미 있었다. 그렇다면 의문이 생겼다 — **이렇게까지 만들었는데도 왜 학생들은 잘 가지 않을까? 무엇을 바꾸면 가게 될까?** '있는 것'과 '이용하는 것'은 다르다. 어쩌면 문제는 공간의 수가 아니라, **만들 때 청소년이 원하는 것이 충분히 반영되지 않은 것** 아닐까?

### 2. 탐구 목적
대치동 청소년 전용공간을 대상으로 ①시설 현황 ②학생의 인지·이용·미이용 이유 ③학생이 원하는 공간을 조사하고, 개선 전후 이용률 변화를 분석한다. 청소년에게 필요한 것은 '더 많은 시설'이 아니라 **청소년 수요에 맞아 실제로 이용되는 생활권 여가공간**임을 데이터로 보인다.

### 3. 이론적 배경 (선행연구)
- **(1) 맞는 공간이 필요한데 '있어도 잘 안 간다'**: 중학생 다수가 청소년 여가시설을 이용하지 않았다(하상희·박정아, 2016). → 청소년이 **진짜 원하는 공간**부터 봐야 함(이연옥, 2025: 공공도서관 청소년 전용공간 요구).
- **(2) 여가공간 → 행복·성장**: 놀이·여가 환경이 좋을수록 청소년 심리적 행복이 높아짐(권민혁, 2025). 중학생 평일 학습시간은 7:34(2024 생활시간조사)로 매우 길어 자유시간이 짧음 → 공간의 '질'이 더 중요.
- **(3) 좋은 공간 = 청소년 의견을 반영해 만든 공간**: 청소년을 주인으로 보고 그 요구를 설계에 반영해야 실제로 쓰인다(이우진, 청소년문화공간디자인). 즉 **의사결정에 청소년 의견이 반영됐는가**가 관건.
- **(4) 디지털 시대에도 공간이 필요할까**: 숏폼 등 디지털 여가가 일상(2024 매체이용 실태조사, 숏폼 94.2%)이고 디지털 여가도 나름의 가치가 있음(이상경 외, 2016). 그래서 오프라인 공간 수요를 설문으로 직접 확인(문10).

## Ⅱ. 연구문제 · 가설
(연구문제 1~6 / 가설 H1~H4 — 1차 보고서와 동일)
- H1 전용시설이 있지만 학생 인지·이용은 낮을 것 (문4)
- H2 미이용 이유는 출입·노출·시간·컨셉 (문5)
- H3 원하는 건 활동+휴식+재미 복합공간 (문6·7)
- H4 디지털이 편해도 오프라인 공간은 원함 (문10)

## Ⅲ. 연구 방법
- 설문: 대명중 중학생 **104명**, 구글폼 10문항, 2026-06-10~06-20
- 공식통계: 강남구청·언론(개선 전후 이용률)
- 선행연구: 국가통계(청소년종합실태조사·생활시간조사)
- 응답자: 중1 34명·중2 38명·중3 32명 / 남 49·여 53·응답안함 2 〔가상〕

## Ⅳ. 결과 1 — 대치동 청소년 전용공간 현황 (공급)
대치동에 청소년 공간이 '없는' 게 아니다. 강남구는 전국 최초·유일 청소년 전용시설을 집중 조성했다.
1. **사이'쉼'**(도곡로 420, 전국 최초 학원가 심리쉼터, 무료) — 놀이·휴식(북카페·보드게임)+전문 심리상담. 누적 5만 명, 연 1.28만~1.5만, 만족 95%+.
2. **스트레스 프리존**(도곡로 1.5평×5부스 + '방과 후 아지트') — 전국 유일 가로형 무인쉼터, 주민참여 리빙랩 기획.
3. **대치4동 '청소년 꿈의 거리'**(한티역~은마 600m, 조성 중).
→ 전용시설이 **이미 존재**한다. 그래서 핵심 질문은 "있는데도 왜·어떻게 이용되는가".

## Ⅴ. 결과 2 — 학생 설문 (인지·미이용·수요) 〔가상 데이터〕

### (1) 자유시간에 가는 곳 [문3]
| 장소 | 비율 |
|------|------|
| 카페·음식점 | 32% |
| PC방·노래방 | 24% |
| 집 | 22% |
| 공원·야외 | 9% |
| 학원가 쉼터·길거리 | 7% |
| 도서관·스터디카페 | 4% |
| 운동장·체육시설 | 2% |
→ 청소년 전용공간은 선택지에 거의 없음. 상업·집 중심.

### (2) 전용시설 인지·이용 [문4] → **H1 검증**
| 시설 | 이용해 봤다 | 들어봤지만 안 감 | 처음 들었다 |
|------|----------|----------------|------------|
| 사이'쉼' | 12% | 31% | **57%** |
| 스트레스 프리존 | 18% | 40% | 42% |
| 청소년 꿈의 거리 | 5% | 19% | **76%** |
→ **전국 최초 시설인데도 '처음 들었다'가 과반(사이쉼 57%·꿈의거리 76%).** 실제 이용은 12~18%에 그침 → **H1 지지**: 공급은 됐으나 인지·이용이 낮다.

### (3) 잘 안 가는 이유 [문5, 1~5점 평균] → **H2 검증**
| 이유 | 평균 |
|------|------|
| 학교–학원–학원–집 동선상 시간이 없다 | **4.1** |
| 무엇을 하는 곳인지 모르겠다 | **3.8** |
| 통유리라 노출돼 부담된다 | 3.6 |
| 차라리 카페·PC방이 편하다 | 3.5 |
| 출입·신청 절차가 번거롭다 | 3.3 |
| 컨셉이 내가 원하는 쉼과 안 맞는다 | 3.2 |
→ '시간 부족'과 '인지 부족'이 최상위, 그다음 '노출 부담' → **H2 지지**. (이 항목들은 2023년 언론이 지적한 실제 미이용 이유와 일치)

### (4) 원하는 공간 [문6·문7] → **H3 검증**
**가고 싶은 시설(복수응답·최대 3개)** [문6]
| 시설 | 선택률 |
|------|------|
| 무료·저렴한 카페형 휴식공간 | 61% |
| VR·e스포츠 등 체험공간 | 48% |
| 운동·클라이밍 등 활동공간 | 35% |
| 보드게임·친구 대화 공간 | 31% |
| 조용히 쉴 휴식공간 | 27% |
| 만들기·창작공간 | 19% |
> (한 명이 최대 3개 선택 → 선택률 합은 100%를 넘을 수 있음)

**요소 중요도(1~5점 평균)** [문7]
| 요소 | 평균 |
|------|------|
| 무료·저렴 | 4.5 |
| 친구와 함께 | 4.4 |
| 그냥 편하게 쉼 | 4.2 |
| 재미 요소(VR·게임 등) | 4.0 |
| 운동·체험 활동 | 3.7 |
| 만들기·창작 | 3.1 |
→ 단순 휴식이 아니라 **무료+친구+휴식+재미가 결합된 복합공간**을 원함 → **H3 지지**.

### (5) 이용료·이용 의향 [문8·문9]
- 지불의향: 무료여야 38% / 1~2천원 34% / 3~5천원 16% / 활동따라 9% → **무료~저렴(1~2천원)이 72%**.
- 이용 의향: 주2~3회 38% / 주1회 27% / 주4회+ 19% → **맞는 공간이면 주 1회 이상 가겠다 84%**.

### 국가통계로 뒷받침
- 2023 청소년종합실태조사(여가부): 하고 싶은 활동 1위 **문화·예술 43.6%**(모험 13.5%·과학 12.5%) → 체험·활동 욕구. 삶 만족 6.97/10. 〔보도자료 원문 확인〕
- 2024 생활시간조사(통계청, 학교급별): **중학생 평일 학습시간 7시간 34분, 문화·여가활동 3시간 7분** → 공부 비중이 커 자유시간이 짧으니, 그 시간을 보내는 **공간의 질**이 중요. 〔원본 엑셀 확인〕

## Ⅵ. 결과 3 — 무엇을 바꾸자 이용이 늘었나 (개선 전후) ★핵심 [공식 통계]
| 구분 | 초기(2023 상반기) | 개선 후(2023.9~) |
|------|------------------|------------------|
| 출입 | 스티커·신청 | 모바일 QR 즉시 출입 |
| 사생활 | 통유리(노출) | 불투명 시트 |
| 운영시간 | 14~21시 | ~22:30·주말 개방 |
| 콘텐츠 | 단순 부스 | 예술전시 '방과 후 아지트' |
| **이용** | 일평균 90명/부스당 40명 | **150명(1.5배)/80명(2배)** |
- 만족 86.4%, 재이용 87.7%(주 요인: 무료 냉난방·충전).
→ 시설을 더 짓지 않고 **이용방식·환경을 청소년에 맞추자 이용 1.5~2배**. 흥미롭게도 **우리 설문의 미이용 이유(출입·노출·시간)와 정확히 같은 지점**을 고쳤더니 이용이 늘었다 → "수가 아니라 맞춤"의 직접 증거.

## Ⅶ. 결과 4 — 수요-공급 불일치
| 축 | 공급(현 공간) | 수요(학생 희망) |
|----|---------------|----------------|
| 성격 | 학습형·소비형·단순휴식 | 활동+휴식+재미 복합 |
| 비용 | 유료 많음 | 무료·저렴(72%) |
| 활동성 | 정적 | 활동·체험(VR 54%) |
| 분위기 | 어른·학습 중심, 노출 부담 | 친구와·눈치 없이 |

**공간 불일치(가상 예시):** 학생이 원하는 'VR·체험형' 54% vs 실제 그런 공간 거의 0 → 큰 불일치. '무료 휴식형' 68% 희망인데 현실은 유료 카페 중심.
→ **결론 문장: 청소년은 놀 공간이 없는 게 아니라, 있는 공간이 맞지 않는다.**

### (반론 검증) 디지털이 편한데 공간이 의미 있을까? [문10]
실제로 청소년의 디지털 여가는 압도적이다 — 2024 매체이용 실태조사에서 숏폼 이용 94.2%, 생성형 AI 약 절반. 그렇다면 물리 공간이 무의미하다는 반론이 가능하다.
- "폰·게임이 편해도 친구와 만날 오프라인 공간은 필요하다" 동의 **평균 4.0/5** (그렇다+매우그렇다 73%) → **H4 지지**.
→ 디지털이 편해도 **오프라인 공간 수요는 분명히 남는다.** 단, 혼자 노는 곳이 아니라 **친구와 함께·디지털도 즐기는** 형태여야 함.

### (보조 논의) 대도시 '풍족 속 불일치' vs 지방 '결핍'
대치동은 시설이 **있어도 맞지 않아** 안 가는 '풍족 속 불일치(질)'. 문화시설이 적은 중소·소도시는 **선택지 자체가 없어 못 가는 '결핍(양)'**일 가능성(국가지표·문화기반시설 지역격차 연구). 같은 결론(생활권 여가공간 부족), 다른 처방 — 대도시는 질, 지방은 양. (공공자료 기반, '가능성'으로 신중히)

## Ⅷ. 결론 및 제언
**결론:** 대치동은 전국 최초 청소년 전용공간을 갖췄지만 초기엔 외면받았고, **청소년 눈높이(즉시 이용·사생활·시간·재미)에 맞추자 이용이 1.5~2배** 늘었다. 우리 설문도 같은 방향 — 학생들은 시설을 **모르거나(처음 들음 과반)**, 알아도 **시간·노출·컨셉** 때문에 안 가며, 원하는 건 **무료·친구·휴식·재미가 결합된 공간**이다.
> **"청소년에게 필요한 것은 '놀 시간'이 아니라, '실제로 가고 싶은 공간'이다."**

**제언**
1. **질(맞춤) + 청소년 참여**: 청소년 수요(무료·친구·휴식·재미, 부담 없는 구조)를 반영하고 출입·시간·콘텐츠를 청소년에 맞춘다. 특히 **공간을 만들·고칠 때 청소년 의견을 직접 듣는 절차**(설문·참여단)를 둔다. 디지털 세대 특성을 인정해 폰·게임도 즐기며 친구와 만나는 온·오프 결합 공간으로.
2. **인지·홍보**: '처음 들었다'가 과반인 만큼, 학교·온라인으로 **공간 안내·홍보**를 강화한다.
3. **양(확충)**: 시설이 부족한 중소·소도시는 양적 확충도 필요(후속 연구).

## Ⅸ. 한계
- 대치동 사례 중심이라 타 지역 일반화에 한계(중소·소도시는 후속 연구).
- 학생이 원해도 학원·시험·귀가시간 등 현실 제약으로 이용 못 할 수 있다.
