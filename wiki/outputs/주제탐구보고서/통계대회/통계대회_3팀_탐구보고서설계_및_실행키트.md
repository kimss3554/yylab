# 통계활용대회 3팀 탐구보고서 설계 & 실행 키트

> 흐름: **탐구보고서(1차) → 발표포스터**. 보고서가 완성돼야 포스터의 그림·문구가 나온다.
> 현실성 원칙: 대상이 **중학생 2팀 + 초등 1팀**. 남은 기간(중등 ~6/27, 초등 ~7/4) 안에 **직접 할 수 있는 통계만** 설계. 빈도·교차표·간단 그래프·지도가 주력. 회귀/머신러닝은 넣지 않음(감점 요인이자 비현실적).
> 후속실천은 보고서 '제언'에서 끝내지 않고 **실제 행동 1개**(캠페인·제안서·안내문)를 작게라도 실행해 사진/결과를 보고서에 넣는다.

## 공통 탐구보고서 표준 목차 (28회 평가기준 정렬)
| 절 | 내용 | 평가 연결 |
|----|------|-----------|
| 1. 주제·동기 | 생활 속 관찰에서 출발한 이유 | 창의성·주제(30) |
| 2. 목적·연구문제(가설) | 답할 질문 1개로 압축 | 논리(30) |
| 3. 선행연구·배경 | 책·기사·기존 통계 2~3개 | 논리 |
| 4. 탐구 방법 | 설문/관찰/공공데이터 + 분석방법 | 통계기법(20+5) |
| 5. 탐구 결과 | **표·그래프·지도(핵심)** | 통계기법·표현(15) |
| 6. 결론·제언 | 가설 검증 결과 + 해석 | 논리 |
| 7. 후속 실천·활용 | **실제 행동 1개 + 결과** | 창의성·표현 |
| 8. 참고문헌·부록 | 설문지 원본 등 | 표현 |

## 1팀. 국민서·이가현 (중) — "친구 생일선물, 마음일까 가격일까?"

### 보고서 절별 설계
1. **동기**: SNS 고가 선물 인증, "얼마짜리 줘야 하지?" 고민 → 또래 부담.
2. **연구문제(3개로 압축)**: ①선택 기준은 가격인가 의미인가 ②친할수록 더 비싸게 주는가 ③선물 부담이 관계 부담으로 이어지는가.
3. **배경**: 사회적 비교 이론·또래압력(책/기사 인용) + 청소년 용돈 국가통계 1개.
4. **방법**: 자체 설문 150~200명 + **인식vs행동 실험문항** + 통계청 청소년 통계(용돈) 인용.
5. **결과**: 빈도(평균 선물금액·선택기준), 교차표(친밀도×금액), 막대/원그래프, **실험 비교 막대**(같은 선물 싼값 vs 비싼값일 때 부담감 차이).
6. **결론**: "말로는 마음, 행동은 가격" 괴리 수치로 제시.
7. **후속 실천**: "선물은 마음" 카드뉴스/포스터 제작 → 반 게시 → **사전·사후 미니설문(10명)**으로 인식 변화 측정 → 결과 1줄 그래프.
8. 부록: 설문지.

### 단계별 To-do (중등, 6/22 설문마감)
- [ ] 6/9까지: 가설 확정 + **설문지 완성**(아래 부록 A)
- [ ] 6/10~6/22: 구글폼 배포(반·학년 단톡), 100명+ 목표
- [ ] 6/16~6/22: 응답 마감 → 구글시트 자동집계 → 그래프
- [ ] 6/23~6/27: 보고서 초안 + 후속 캠페인 실행(카드뉴스 1장)
- [ ] 7월: 시험 후 교정·포스터화

### 현실성 체크 ✅/✂️
- ✅ 설문·빈도·교차표·실험문항: 중학생이 구글폼+구글시트로 가능.
- ✂️ 회귀분석 빼기. 상관은 "용돈 많을수록 금액↑" 산점도 1개로 가볍게.
- ✂️ 표본 300명 무리 → **100~150명**으로 현실화.

## 2팀. 홍정인 (중) — "우리 동네 청소년은 어디서 노는가" (여가공간)

### 보고서 절별 설계
1. **동기**: 갈 곳 없어 노는 청소년, 동네별 시설 차이 체감.
2. **연구문제**: ①청소년은 주로 어디서 여가를 보내나 ②여가시설 만족도와 행복은 관계있나 ③동네별 시설 분포는 공평한가.
3. **배경**: 국민여가활동조사·생활시간조사(청소년 여가시간) 인용 + 청소년시설 현황.
4. **방법**: 자체 설문(이용장소·만족/중요도 IPA·행복) + **공공데이터 시설좌표 → 지도**.
5. **결과**: 빈도(주 이용장소), **IPA 사분면 그래프**(개선 우선순위), **여가시설 분포 지도**(My Maps), 동네별 시설수 비교.
6. **결론**: "시설 적은 동네일수록 만족·행복 낮음" 등.
7. **후속 실천**: '청소년 여가공간 개선 제안서' 작성 → **구청/학교에 실제 건의(민원·메일)** → 회신/접수증 첨부.
8. 부록: 설문지 + 데이터 출처.

### 단계별 To-do
- [ ] 6/9까지: 설문지 완성(부록 B), 거주 동네 범위 확정
- [ ] 6/10~6/15: 설문 배포 + **공공데이터포털 시설좌표 내려받기**(아래 부록 C)
- [ ] 6/16~6/22: 설문마감·분석, **My Maps 분포지도** 제작, IPA 그리기
- [ ] 6/23~6/27: 보고서 초안 + 제안서 발송(후속실천)
- [ ] 7월: 교정·포스터화

### 현실성 체크 ✅/✂️
- ✅ 설문·IPA·My Maps 핀 찍기: 중학생 가능(IPA는 평균 2개 찍어 사분면 — 지도교사 1회 시연).
- ✂️ **QGIS·SGIS 고급 접근성 분석은 선택/생략**. 기본은 "동네별 시설 수 세기 + My Maps 분포 + 만족도 비교"로 충분.
- ✂️ 시설 종류 너무 넓히지 말 것 → 청소년수련관·도서관·공원·체육시설 4종으로 한정.

## 3팀. 배이아 (초등) — "우리 동네 새 지도: 우장산의 새들"

### 보고서 절별 설계 (초등 눈높이·문장 쉽게)
1. **동기**: 『새들의 밥상』 읽고 집 근처(우장산) 새소리에 관심, 소쩍새 발견.
2. **탐구 질문**: 우리 동네엔 어떤 새가, 언제, 어디에 많이 나타날까?
3. **배경**: 책 내용 + 네이처링/eBird에서 우장산 일대 새 기록 살펴보기.
4. **방법**: **BirdNET 앱**으로 2~4주 관찰·기록(날짜·시간·장소·새·마릿수·날씨).
5. **결과**: 빈도(제일 많이 본 새 막대그래프), 시간대·장소별 표, **우리 동네 새 지도**(My Maps 핀), 책/eBird와 비교.
6. **결론**: "산과 가까운 곳에 새가 더 많다" 등 �운 발견 + 천연기념물 소쩍새 이야기.
7. **후속 실천**: '새를 지켜요' 안내문/포스터 만들어 학교·아파트 게시, 산 쓰레기 줍기 1회 → 사진.
8. 부록: 관찰기록표 원본.

### 단계별 To-do (초등, 시험 없음 → 길게)
- [ ] 6/12까지: 관찰표(부록 D) 준비, BirdNET·My Maps 세팅, 관찰 시작
- [ ] 6/13~6/27: **관찰 집중**(아침·저녁, 산쪽·도시쪽 등 장소 다양화), 매번 기록
- [ ] 6/28~7/4: 관찰 마무리, 막대그래프·새 지도 완성, 보고서 작성, 안내문 게시(후속)
- [ ] 7/5~7/8: 다듬기·제출

### 현실성 체크 ✅/✂️
- ✅ BirdNET 기록·막대그래프·My Maps: 초등 가능(보호자/지도 동행 권장).
- ✅ 시험 없음 → 관찰 기간 길게 = 데이터 강점.
- ✂️ 통계기법 욕심 금지. **빈도·그림그래프·지도**면 초등부 충분. 종 다양성 지수 등 불필요.
- ⚠️ 안전: 산 관찰은 보호자 동행, 정해진 산책로에서만.

# 부록 A. 국민서·이가현 설문지(초안)

*구글폼. 5점 척도 = 1전혀아니다~5매우그렇다*

1. 성별 / 2. 학년 / 3. 한 달 평균 용돈(원)
4. 최근 1년 친구 생일선물에 보통 쓴 금액: ①5천 미만 ②5천~1만 ③1~2만 ④2~3만 ⑤3만 이상
5. 선물 고를 때 가장 중요한 것: ①가격 ②실용성 ③친구 취향 ④내 마음 표현 ⑤남들 시선
6. (5점) 나는 선물의 '가격'을 중요하게 생각한다
7. (5점) 나는 선물의 '의미·마음'을 중요하게 생각한다
8. 친한 친구일수록 더 비싼 선물을 준다: ①그렇다 ②보통 ③아니다
9. (5점) 친구 선물 살 때 금액이 부담된다
10. (5점) 비싼 선물을 받으면 비슷하게 갚아야 할 것 같다
11. (5점) 선물 때문에 친구관계가 스트레스가 된 적 있다
12. SNS에서 친구 선물 인증을 본 뒤 더 신경 쓴 적 있다: 예/아니오
13. **[실험]** 같은 텀블러입니다. A: 5,000원 / B: 25,000원.
    13-1. A를 받으면 보답 부담? (5점) 13-2. B를 받으면 보답 부담? (5점)
14. (5점) 나는 용돈을 계획적으로 쓴다
15. 자유의견(선물문화에 하고 싶은 말)

> 분석: Q5 빈도, Q6vsQ7 평균비교, Q8 교차, Q9~11 상관/평균, **Q13-1 vs 13-2 막대비교(핵심)**, Q14로 경제관념 상·하 나눠 비교.

9. **[IPA]** 아래 각 시설의 (A)중요도 / (B)현재 만족도를 각각 5점으로:
   청소년수련관 / 도서관 / 공원 / 체육시설 / 카페·쉼터

# 부록 C. 홍정인 — 공공데이터 시설좌표 받기 (단계별)

1. **data.go.kr** 접속 → 검색창에 "청소년수련시설" / "공공도서관" / "도시공원" / "공공체육시설" 검색.
2. 각 결과에서 **CSV 또는 표준데이터** 다운로드(시설명·주소·위도·경도 포함된 파일 선택).
3. 서울 거주면 **data.seoul.go.kr**에서 자치구별 파일이 더 정확.
4. 엑셀에서 우리 동네(구) 행만 남기고 정리.
5. **Google My Maps**(google.com/maps/d) → '새 지도' → '가져오기'로 CSV 업로드 → 위도·경도 열 지정 → 시설 종류별 색 핀.
6. (선택·심화) **SGIS** sgis.kostat.go.kr → '활용서비스 > 통계주제도'에서 인구 대비 시설 보기.
7. 결과 지도 + 설문 만족도 = "시설 많은 동네 vs 적은 동네" 비교.

> 좌표 없는 파일이면 주소만으로도 My Maps가 자동 위치표시 → 초보는 이게 더 쉬움.
