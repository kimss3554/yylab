# 통계활용대회 3팀 주제 설계 & 준비 일정

> 마감: **2026-07-08(자료 완성)** / 중·고 시험: **7/1~7/3** → 중등팀은 6월 말부터 참여 어려움 / 초등(배이아)은 시험 없음
> 수상 전략 원칙(수상작 경향 분석 기준): **명확한 단일 질문 + 자체 데이터 생산 + 기본 시각화 + 후속 실천**. 회귀·머신러닝 등 고급기법은 보조로만(감점 요인).

## 0. 공통 데이터 소스 지도 (어디서 가져오나)

| 포털 | URL | 무엇을 | 누구에게 |
|------|-----|--------|----------|
| KOSIS 국가통계포털 | kosis.kr | 인구·청소년·여가·소비 등 거의 모든 국가통계 표 | 전 팀 |
| **통계청 SGIS 통계지리정보** | sgis.kostat.go.kr | **지도 위 통계·생활권 분석·시설 접근성·인구격자** | 홍정인(핵심), 배이아 |
| 공공데이터포털 | data.go.kr | 시설 위치(좌표) CSV, 각종 행정데이터 | 홍정인, 배이아 |
| 서울 열린데이터광장 | data.seoul.go.kr | 자치구별 시설·청소년·생활 데이터 | 서울 거주 시 |
| e-나라지표 | index.go.kr | 지표 요약·추세 그래프(인용 쉬움) | 전 팀 |
| Google My Maps | google.com/maps/d | 학생이 직접 점 찍어 지도 만들기(쉬움) | 전 팀(특히 초등) |

지도 시각화 도구: **쉬움→** Google My Maps / 카카오맵·네이버 '내 지도' → **중간→** SGIS 활용 → **고급→** QGIS(중등 심화팀만 선택).

## 1. 국민서·이가현 (중·고) — "친구 생일선물, 마음일까 가격일까?"

### 주제 다듬기 (수상 포인트로)
- 단일 핵심 질문: **"청소년은 선물에서 '마음'이라 말하지만 실제로는 '가격'에 얼마나 좌우되는가?"**
- 차별화 무기 = **인식 vs 행동 실험**: 동일 선물을 다른 가격표로 제시 → "말과 행동의 괴리"를 수치로 보여줌(대상작이 좋아하는 '검증 설계').
- 후속 실천: "건강한 선물문화" 캠페인 + 사전·사후 인식 변화 측정.

### 자체 설문 설계(주 데이터)
- Google Forms, 중·고생 **150~300명**. 문항: 성별/학년/월용돈/친밀도/실제 선물금액/선택기준/부담감 척도(5점)/경제관념.
- 실험문항: 같은 텀블러를 (A)5,000원 (B)25,000원으로 보여주고 "받았을 때 부담/보답의무" 응답 비교.
- 분석: 빈도 → 교차(친밀도×금액) → 상관(가격중시↔부담감, 용돈↔금액) → 집단비교(경제관념 상/하). 회귀는 "부담감 영향요인" 보조용 1개만.

### 국가통계·2차 자료 (배경·신뢰도 보강)
| 무엇 | 어디서 |
|------|--------|
| 청소년 용돈·소비 실태 | 여성가족부 **청소년종합실태조사**, 통계청 **청소년 통계**(매년 5월), 통계청 **사회조사**(소비생활) |
| 또래·SNS 영향 | 한국언론진흥재단 **10대 청소년 미디어이용 조사**, KISDI 보고서 |
| 소비자 관점 | 한국소비자원 청소년 소비 실태 자료 |

### 지도/분포도 활용
- 이 주제는 지리 성격이 약함 → **지도 대신 인포그래픽·교차표 히트맵**에 집중 권장.
- 굳이 쓴다면: 여러 학교/지역 표본일 때 "지역별 평균 선물금액" 비교를 SGIS나 색칠지도로. (선택)

## 2. 홍정인 (중·고) — "우리 동네 청소년은 어디서 노는가" (여가공간)

### 주제 다듬기
- 단일 핵심 질문: **"생활권 여가환경의 양·질이 청소년 행복/만족을 얼마나 좌우하며, 그 환경은 동네별로 공평한가?"**
- 차별화 무기 = **'여가시설 격차 지도'**: 동네별 청소년시설 분포 + 만족도를 겹쳐 "여가 사막(desert)" 시각화. → 세 팀 중 지도/분포도 활용이 가장 강력.

### 자체 설문(주 데이터)
- 청소년 대상, 주 이용 장소·이동시간·만족/중요도(**IPA 중요도-성취도 분석**), 행복척도.
- 분석: 빈도 → 교차 → IPA 사분면(개선우선순위 시각화) → (보조)동네별 만족도 비교.

### 국가통계·2차 자료
| 무엇 | 어디서 |
|------|--------|
| 청소년 여가시간·활동 | 통계청 **생활시간조사**, 문화체육관광부 **국민여가활동조사**, 통계청 **청소년 통계** |
| 청소년시설 현황 | 한국청소년활동진흥원 청소년수련시설 통계, 여성가족부 |

### 지도/분포도 활용 (이 팀의 핵심)
1. **시설 위치 데이터 받기**: 공공데이터포털(data.go.kr)에서 "청소년수련관/문화의집/공공도서관/공원/체육시설" 위치 CSV(좌표 포함). 서울이면 서울 열린데이터광장 자치구별.
2. **SGIS(sgis.kostat.go.kr)**: 인구(청소년 인구 격자) 대비 시설 수, 생활권 분석, 접근성. → "청소년 1,000명당 시설 수" 동네별 지도.
3. **지도 제작**: 받은 좌표를 Google My Maps에 올려 분포도 → 설문 만족도와 겹쳐 해석. 심화 시 QGIS로 도보 접근권(반경 500m) 분석.
4. 결론 시각화: "시설 많은 동네 vs 적은 동네 × 만족도/행복" 대비 지도 1장이 포스터의 hero 이미지.

## 3. 배이아 (초등) — "우리 동네 새 지도: 우장산의 새들" (시민과학)

### 주제 다듬기 (초등 눈높이)
- 단일 핵심 질문: **"우리 동네(우장산)에는 어떤 새가, 언제, 어디에 많이 나타날까?"**
- 차별화 무기 = **직접 만든 시민과학 데이터 + 천연기념물 소쩍새 발견 스토리**(도시 옆 산에 천연기념물!). 초등부에서 '직접 관찰·기록'은 매우 강한 강점.

### 자체 관찰 데이터(주 데이터) — 초등 적합
- **BirdNET 앱**으로 날짜/시간/장소/새종류/(소리)를 표로 기록. 2~4주 누적.
- 기록 항목: 날짜·시간대·관찰장소·새 이름·마릿수·날씨. → 엑셀/공책 표.
- 분석(초등 수준): **빈도**(어떤 새가 제일 많이?), 시간대·장소별 출현, **그림그래프·막대그래프**. 회귀 등 고급기법 불필요(오히려 감점).

### 공공·시민과학 자료 (배경·비교)
| 무엇 | 어디서 |
|------|--------|
| 우리나라 조류 분포·관찰기록 지도 | **네이처링(naturing.net)**, **eBird Korea(ebird.org)** — 관찰 점이 지도에 찍힘(분포도 그대로 활용) |
| 전국 조류 개체수·종 | 국립생물자원관(nibr.go.kr), 환경부 **겨울철 조류 동시센서스**, 국가생물다양성 정보공유체계 |
| 천연기념물·멸종위기 정보 | 문화재청, 국립생물자원관 적색목록 |
| 날씨(출현과 연결) | 기상청 |

### 지도/분포도 활용 (초등 친화)
1. **Google My Maps**에 본인 관찰 지점(우장산 산책로별)에 새 이름 핀 찍기 → '우리 동네 새 지도' 완성.
2. **네이처링/eBird** 우장산·강서구 일대 검색 → 다른 사람 관찰과 내 관찰 비교(내가 찾은 소쩍새가 기록에 있나?).
3. 결론: "산과 가까운 곳일수록 새 종류가 많다" 같은 단순·명확한 발견 + 새 보호 실천 제안(쓰레기 안 버리기).

> 배이아는 시험이 없으므로 **6월 말~7월 초까지 관찰을 계속**해 데이터를 더 쌓을 수 있음(다른 두 팀과 일정 분리).

## 4. 통합 준비 일정 (오늘 6/5 기준, 마감 7/8)

### 트랙 A — 중등 2팀(국민서이가현·홍정인): **설문은 시험 전에 닫는다**
| 기간 | 할 일 |
|------|-------|
| 6/5~6/9 | 주제·핵심질문·가설 확정, **설문지 제작**(실험문항·IPA 포함), 2차자료 1차 수집 |
| 6/10~6/15 | **설문 배포·수집**(SNS·학교), 시설 좌표/국가통계 내려받기 |
| 6/16~6/22 | **설문 마감**, 통계분석, 지도(SGIS/My Maps) 제작 |
| 6/23~6/27 | 결과 해석, **포스터/보고서 초안** + 핵심 시각화 완성 (학생 참여 사실상 여기까지) |
| 6/28~7/3 | **시험기간** — 학생 최소 관여. 지도교사/멘토가 문구·디자인 다듬기, AI 사용 기술서 작성 |
| 7/4~7/8 | 최종 검토·교정·**제출** |

핵심: 중등팀 **설문은 6/22까지 반드시 마감**(시험 전 분석 끝내기). 6/9 전 설문지 확정이 관건.

### 트랙 B — 초등 1팀(배이아): 여유 있게, 관찰을 길게
| 기간 | 할 일 |
|------|-------|
| 6/5~6/12 | 관찰 항목표 확정, BirdNET 관찰 시작, Google My Maps 세팅 |
| 6/13~6/27 | **관찰·기록 집중**(시간대·장소 다양화), 네이처링/eBird 비교 |
| 6/28~7/4 | (시험 없음) 관찰 마무리, 빈도·그래프·새 지도 완성, 보고서 작성 |
| 7/5~7/8 | 다듬기·**제출** |

### 분기점(모두 동일)
- 6/9: 중등 설문지 최종본 OK? (아니면 일정 압박)
- 6/22: 중등 설문 마감·데이터 확보 완료?
- 6/27: 중등 초안+시각화 완성? (시험 들어가기 전 마지노선)

## 5. 팀별 '지도/분포도' 한눈에
- **홍정인**: SGIS + 공공데이터포털 시설좌표 → "여가시설 격차 지도"(가장 강력)
- **배이아**: Google My Maps + 네이처링/eBird → "우리 동네 새 지도"
- **국민서이가현**: 지도 비중 낮음 → 인포그래픽·교차표 히트맵 권장
