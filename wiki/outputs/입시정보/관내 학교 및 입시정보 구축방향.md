# 관악구 맞춤형 입시정보 시스템 구축 마스터플랜

## 1️⃣ 비전 & 전략

### 목표
**관악구 지역 기반 전 단계적 진학 네트워크 구축**
- 초등학교부터 대학입시까지 연속성 있는 진학 로드맵 제공
- 지역 내 11개 고등학교 중심의 맞춤형 입시정보 독점화
- 학부모 의사결정 지점(critical moments)에서의 전략적 개입

### 경쟁 차별화 요소
- ✅ **지역 특화**: 광범위한 정보가 아닌 "관악구 11개 고교"에 대한 심화 정보
- ✅ **연계성**: 초등→중등→고등 이상의 연속 컨설팅 (일관된 담당자)
- ✅ **프로세스 기반**: 일회성 정보 아닌 단계별 체계적 가이드
- ✅ **정량 + 정성**: 입시결과 데이터 + 학교문화/선생님 특성 등 정성정보

### 학교별 데이터 수집 항목

```
📊 입시 데이터 (지난 3-5년)
├─ 수시/정시 진학 비율
├─ 대학 진학 현황 (SKY, 중상위권, 지거국 비율)
├─ 평균 수능 성적 추이
├─ 학과별 입시 결과 (자연/인문/예체능)
├─ 특목고/자사고 전입생 현황
└─ 재수/반수 비율

🎓 교육과정 특성
├─ 필수 교과목 구성
├─ 2-3학년 선택과목 체계
│  ├─ 선택 가능한 조합
│  ├─ 학생들의 선택 추이
│  └─ 평가원 모평에서의 난이도 추세
├─ 절대평가 vs 상대평가 비율
├─ 각 과목별 수행평가 배점 구조
└─ 탐구보고서, 실험 중심 교과

🏆 비교과 활동 생태계
├─ 동아리 규모 & 구성
├─ 교내 대회/발표 기회 (과목별)
├─ 진로활동 프로그램
├─ 세특 기재 관행
│  ├─ 적극형 학교 vs 보수형 학교
│  ├─ 과목별 평균 글자수
│  └─ 자주 기재되는 활동 유형
└─ 학생회/리더십 활동 수준

👥 학교 특성
├─ 학생 구성 분석
│  ├─ 지역 분포 (강남/강북 비율)
│  ├─ 사회경제적 배경
│  └─ 학력 수준 대비 입시 성과
├─ 교사진 특성
│  ├─ 입시담당 선생님 경력
│  ├─ 진로지도 수준
│  └─ 과목별 명강사 여부
├─ 학교 이슈 (상승/하강 추세)
└─ 최근 3년 변화 추이
```

## 3️⃣ 의사결정 포인트별 컨설팅 전략

### 📚 초등학교 (4-6학년)

**핵심 의사결정**: "중학교를 어디로 가야 할까?"

| 시기 | 의사결정 내용 | 우리의 역할 |
|------|----------------|-----------|
| **4학년** | 특기 발굴 및 학습 유형 파악 | 아이 강점 진단 + 관악구 중학교 특성 설명 |
| **5학년** | 영재원/특수반 지원 여부 결정 | 입시 경로 시뮬레이션 |
| **5-6학년 상반기** | 중학교 유형 선택 (자사중 vs 일반중) | 관악구 내 중학 선택지 분석 및 컨설팅 |

**제공 서비스**
- 🎯 아이별 강점 진단 (과목별 적성, 학습 스타일)
- 📖 독서토론 & 사고력 확장 수업
- ✍️ 글쓰기 & 표현력 워크숍
- 💡 "우리 아이에게 맞는 중학교 선택" 1:1 상담

---

### 🎯 중학교 (1-3학년)

**핵심 의사결정**: "고등학교를 어떻게 준비할 것인가?"

| 시기 | 의사결정 내용 | 우리의 역할 |
|------|----------------|-----------|
| **입학 전 (1월)** | 관악구 고등학교 조사 시작 | "11개 고교 지도" 제공 및 설명 |
| **1학년 상반기** | 관악고 진학 가능성 판단 | 성적 추이 분석 + 목표 고교 1차 선정 |
| **1학년 하반기** | 고교 진학 방향 최종 결정 (일반고 vs 특목고) | 입시컨설팅 + 목표 설정 |
| **2학년** | 세특 전략 수립 | "고3 입시에 도움될 활동" 설계 |
| **2-3학년 상반기** | 특목자사고 입시 준비 | 자기소개서 및 면접 컨설팅 |

**제공 서비스**
- 📊 **중학 생기부 최적화 가이드** (세특, 활동기록 전략)
- 🎓 **관악구 11개고 상세 분석 설명회** (분기별 1회)
- 🎯 **목표 고교 설정 1:1 상담**
- ✍️ **특목자사고 자기소개서 컨설팅** (개인맞춤 피드백 5회)
- 📚 **대입을 고려한 과목선택 가이드** (수학/과학 심화 경로)
- 💬 **부모특강**: "중학 생활기록부가 대입을 결정한다"

### 🚀 고등학교 (1-3학년) - 가장 중요한 단계

#### 📌 고1 (3월 ~ 2월)

**의사결정 포인트**
- 첫 수행평가 대비 및 과목별 학습전략 수립
- 진로활동의 방향 설정
- 2학년 과목선택을 위한 전반 탐색

**제공 컨설팅**
- ✍️ **"첫 수행평가 완벽가이드"** 
  - 각 교과별 평가 방식 사전분석
  - 서답형, 탐구보고서, 자료분석 형식별 팁
  
- 🎓 **"학교별 세특 기재 문화 가이드"**
  - 구암고/당곡고/... 각 학교의 세특 특성
  - "이 학교에서 높이 평가받는 활동은?"
  - "세특 글자수 기준은?"
  
- 📊 **"4월 모의고사 분석 & 과목선택 예비상담"**
  - 선택과목 추천 (수학, 과학)
  - 동료 학생들의 선택 추이 데이터 제공
  
- 💡 **"비교과 활동 로드맵 설계"**
  - 동아리 선택 전략
  - 고1-3 전 단계 진로활동 계획

---

#### 📌 고2 (3월 ~ 2월)

**의사결정 포인트**
- **과목선택 최종 확정** (4월)
- 수시/정시 대략적 방향 설정
- 2학기 탐구보고서 및 산출물 기획

**제공 컨설팅**
- 🎯 **"과목선택 최종 가이드"** (1월~3월)
  - "당신의 성적이라면 어떤 조합이 유리한가?"
  - 대학 모집요강 선택과목 분석
  - 이전 합격생들의 선택 데이터
  
- 📈 **"세부능력 특기사항(세특) 적재 전략"**
  - 각 과목별 교사와의 상담 포인트
  - "이런 활동을 하면 세특에 기재될 확률이 높다"
  - 실제 합격 사례 공유
  
- 🔬 **"탐구보고서 & 산출물 전략"**
  - 선택과목별 추천 주제
  - 대학 입시에서 평가하는 포인트
  - 실제 작성 예시 제공
  
- 💬 **"대학별 수시 전형 분석"**
  - 학생의 성적 수준에 맞는 대학 추천
  - 각 전형별 합격 확률 계산

---

#### 📌 고3 (1월 ~ 12월)

**의사결정 포인트**
- 수시 vs 정시 최종 선택
- 수시 원서 6개 배치 전략
- 대학별 고사/면접 준비

**제공 컨설팅**

**1월~6월 (상반기)**
- 📊 **"모의고사 기반 입시전략 수립"**
  - 3월/4월/6월 모평 분석 → 수시/정시 선택
  - "당신의 성적이라면 수시 6개 어떻게 배치할까?"
  
- 🎓 **"수시 전형별 전략"**
  - 학생부종합 vs 학생부교과 vs 논술 vs 특기자
  - 각 전형의 합격 가능성 분석
  
- ✍️ **"자소서 컨설팅"** (선택: 대학이 자소서 요구시)
  - 최대 5회 첨삭 및 피드백

**7월~8월 (수시 준비)**
- 📝 **"원서 접수 최전 전략"**
  - 6개 대학 배치 (상향/적정/하향)
  - 학과 선택 전략
  
- 💬 **"대학별 고사 대비"**
  - 학교별 기출 분석
  - 모의 고사 진행

**9월~10월 (수시 시험)**
- 🎤 **"면접 컨설팅"**
  - 대학별 면접 유형 분석
  - 예상 질문 & 답변 준비
  - 실제 모의면접 진행
  
- 📊 **"정시 준비 병행"**
  - 9월 모평 분석 및 정시 목표 설정
  - 남은 100일 수능 대비 로드맵

**11월 (수능 후)**
- 📈 **"최종 전략 회의"**
  - 수시 합격 예측
  - 정시 지원 전략 수립
  - 재학/재수 판단 자료 제공

## 6️⃣ 1:1 컨설팅 체계

### 상담 타입별 구성

**Type A: 초등학부모 상담** (60분, 월 4회 모집)
