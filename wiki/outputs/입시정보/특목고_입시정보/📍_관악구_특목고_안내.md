## 🎯 특목고 입시 결정 포인트

### "우리 아이가 특목고에 갈까, 일반고에 갈까?"

#### 특목고 추천 학생 ✅
```
□ 특정 과목에 탁월한 재능
  (예: 수학, 영어, 과학 최상위)

□ 고난도 학습을 즐기는 성향
  (스스로 고민하고 문제 해결)

□ 명확한 진로 방향
  (의대, 대학 공학과, 외교관 등)

□ 강한 학습 동기
  (부모 강요가 아닌 본인 의지)

□ 체계적인 학습 습관
  (시간관리, 자율학습 가능)
```

#### 일반고 추천 학생 ✅
```
□ 전반적으로 우수하지만 특출난 과목 없음

□ 주변과의 경쟁보다 자신의 페이스 중요

□ 다양한 활동과 경험 원함

□ 진로가 아직 정해지지 않음

□ 정상적인 학교생활 원함
```

### 2️⃣ 국제고 (영어 중심)

#### [[국제고|학교명]]

**기본정보**
- **위치**: 서울시 강남/서초 방향
- **특징**: 국제화 교육, 영어 중심 교과
- **교육목표**: 글로벌 리더 양성

**입시정보**
```
모집정원: 약 ?명
지원자격: 
- 국내 중학교 졸업(예정)자
- 특별 자격 요구 (영어능력 등)

입시방식: 
- 1단계: 내신(?)% + 자기소개서(?)%
- 2단계: 면접

합격선(작년): 
- 내신 1.5~2.5 등급대
- 영어 수능 또는 토플/토익 우수
```

**학교 특성**
```
교육과정:
- 영어 심화 교과 (영어 3시간/주)
- 국제관계, 국제법, 국제경제 등 개설
- 해외 파견 프로그램

강점:
- 영어권 대학 진학 강점
- 국제 활동 기회 풍부
- 글로벌 네트워크

약점:
- 수학/과학 학생 상대적 약점
- 입시 경쟁력이 자사고보다 낮은 편
- 영어 못하는 학생은 고생

진학 현황:
- 국내: SKY 40%, 중상위 40%
- 해외: 미국 명문대 진학 다수
```

**대상학생**
```
✅ 추천:
   - 영어 원어민 수준
   - 국제관계/외교 관심
   - 해외 진학 고려

❌ 비추천:
   - 영어는 평범
   - 이과 강점
```

**우리의 조언**
```
중2-3 준비전략:
1. 영어 최상위 달성 (1등급)
2. 토플/토익 응시 (80+, 700+)
3. 국제관계 관심사 개발
4. 영어 기반 세특 축적

입학 후 전략:
→ 영어는 당연, 수학은 더 강하게
→ 해외 대학 진학 고려 시 추천
```

### 3️⃣ 외고 (특정 언어 중심)

#### [[외고|학교명]]

**기본정보**
```
위치: 서울시 강남/서초/중구 방향
특징: 특정언어(영어/중국어/일본어 등) 심화
교육목표: 언어 전문가 양성
```

**입시정보 (일반적)**
```
모집정원: 약 300-400명
지원자격: 중학 졸업(예정)자
선발방식:
- 1단계: 내신 반영 (50%)
- 2단계: 필기고사 (영어/언어) + 면접

합격선: 내신 1.5~3등급
```

**학교 특성**
```
강점:
- 언어 교육 최고 수준
- 해외 교환학생 프로그램
- 국제화된 학생 분위기
- 국내외 대학 진학 유리

약점:
- 수학/과학은 상대적으로 약함
- 자사고보다 입시난이도 낮음
- 이과 진학 시 경쟁력 저하
- 일부 과목은 수강인원 적음

진학현황:
- SKY: 약 30%
- 중상위대: 약 50%
- 해외 대학: 소수
```

**대상학생**
```
✅ 추천:
   - 특정 언어 원어민 수준
   - 언어학/국제 관계 진학
   - 인문학 강점
   - 해외 대학 고려

❌ 비추천:
   - 이과 진학 희망
   - 언어는 평범한 수준
```

**준비전략**
```
중2-3 중점:
1. 해당 언어 최상위 (1~2등급)
2. 원어민/교포 수준 목표
3. 해당국가 문화/역사 학습
4. 내신 전과목 우수 (2등급 이상)

입학후:
→ 언어 활용 중심의 진로설정
→ 수학/과학은 심화해야 대학 경쟁력 확보
```

### 4️⃣ 영재학교 (수학/과학 최고 영재)

#### [[영재학교|한국과학영재학교]] (서울 강서)

**기본정보**
```
위치: 서울시 강서구
특징: 영재의 조기 발굴 및 교육
수학/과학 중심, 3년 과정 (빠른 입시)
교육목표: 과학기술 리더 양성
```

**입시정보**
```
모집정원: 약 150명 (전국단위)
지원자격: 중2 수료자
입시방식:
- 1단계: 영재성검사 (수학/과학)
- 2단계: 영재성면접

합격선: 상위 0.1% 수준의 수학/과학 천재
```

**학교 특성**
```
강점:
- 최고 수준의 이과 교육
- 영재 학생끼리의 경쟁과 자극
- 창의적 문제해결 중심 교육
- 일찍 졸업 (고2 말에 대입)
- 대학 진학률 거의 100% (상위대)

약점:
- 매우 어려운 입시
- 과도한 학습 강도
- 선발기준 매우 엄격
- 학교생활의 자유도 낮음
- 일반 학생들과 분리된 교육

진학현황:
- SKY 합격률 95% 이상
- 의대/수의대 30% 이상
- 대부분 최상위 대학 진학
```

**입시 준비**
```
초등 고학년부터 준비 필요:
1. 수학 올림피아드 참가 및 입상
2. 과학 영재 프로그램 참여
3. 영재원 선발 준비

중학생 준비:
- 중1: 영재성검사 대비 (고난도 문제)
- 중2: 영재성면접 대비
- 중2 가을: 입시 실시 및 결과
- 중3: 대학 입시 준비 (빠른 졸업)
```

**우리의 조언**
```
⚠️ 매우 신중하게 결정하세요

자녀 확인사항:
□ 수학/과학을 정말 좋아하는가? (강요 아님)
□ 고난도 학습을 견딜 수 있는가?
□ 또래와의 경쟁에 견딜 수 있는가?
□ 학교 규칙에 잘 따를 수 있는가?

YES 3개 이상이면 추천
NO가 하나라도 있으면 신중히 재검토
```

### 5️⃣ 과학고 (과학/수학 심화)

#### [[과학고|관악구 인근 과학고]]

**기본정보**
```
위치: 서울/경기 과학고
특징: 과학기술 인재 양성
수학/과학/정보 중심
교육목표: 과학기술 전문가 양성
```

**입시정보 (일반)**
```
모집정원: 약 150-200명
지원자격: 중학 졸업(예정)자
입시방식:
- 내신 (50%)
- 면접 및 과제면접 (50%)
- 또는 필기고사 (교과내용) + 면접

합격선: 내신 1~2등급 + 수학/과학 최상위
```

**학교 특성**
```
강점:
- 최고 수준의 과학 교육
- 대학 연구실 참여 기회
- 과학 프로젝트 중심 학습
- 대학 진학 강점 (이과)
- 영재학교보다는 상대적으로 부담 적음

약점:
- 수학/과학이 약하면 고생
- 인문학 교육은 상대적으로 약함
- 인문계 진학시 불리
- 높은 학습 강도

진학현황:
- SKY 50%
- 상위 이공계 대학 70%
- 의대/수의대 20% 이상
```

**대상학생**
```
✅ 추천:
   - 수학/과학 최상위 (1등급)
   - 과학기술 진로 확정
   - 고강도 학습 가능
   - 실험/프로젝트 좋아함

❌ 비추천:
   - 수학/과학 평범
   - 인문계 진학
   - 여유로운 학교생활 원함
```

**준비전략**
```
중2-3 중점:
1. 수학/과학 1등급 확보
2. 과학 경시대회 입상 이력
3. 영재원 또는 영재교육센터 이수
4. 실험/탐구활동 적극 참여
5. 세특에 과학 탐구 기재

입학후:
→ 자연계 진학 최적 환경
→ 의대, 공학과 등 강점
```

## 💬 특목고 입시 상담 가이드

### 상담 시 확인사항

**학생 면접**
```
1. "당신이 가장 좋아하는 과목은?"
   → 자발적 답변 확인 (강요받은 것 아닌가)

2. "왜 특목고를 가고 싶어?"
   → 명확한 진로와 연결되어 있는가

3. "이 학교 교육과정을 아니?"
   → 충분히 조사하고 준비했는가

4. "경쟁과 어려움을 견딜 수 있겠니?"
   → 현실적 마인드셋 확인
```

**학부모 면접**
```
1. 자녀의 진정한 의사인가?
   (부모 욕심이 아닌가?)

2. 재정상황 확인
   (특목고는 추가비용 발생)

3. 통학 가능한가?
   (멀면 생활의 질 저하)

4. 대학 진학 비전이 명확한가?
   (그냥 좋은 학교라서? X)
```

## 🎁 우리의 최종 조언

> **특목고의 3가지 진실**

### 1. 특목고가 답은 아니다
```
특목고 진학 = 대학 성공 (X)
특목고에서도 떨어질 수 있다
일반고에서도 SKY 합격할 수 있다
```

### 2. 자녀의 행복이 최우선이다
```
수학을 싫어하는데 과학고? (X)
부모 욕심만 있다면? (X)
자녀가 정말 원하는가? (O)
```

### 3. 현명한 선택이 최고다
```
자녀의 강점을 살리는 학교 선택
장기적 진로와 연결된 선택
지속 가능한 학교 생활 가능한가
```
