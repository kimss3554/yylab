# 관악구 지역 기반 맞춤형 진학 정보 시스템 구축 연구계획서

---

## 1. 연구 개요

| 항목 | 내용 |
|------|------|
| **연구 제목** | 관악구 지역 특화 맞춤형 진학 정보 시스템 구축 및 운영 방안 연구 |
| **연구 기간** | 2026년 5월 ~ 2027년 4월 (12개월) |
| **연구 대상** | 관악구 소재 고등학교 11개교 및 재학 학생·학부모 |
| **연구 유형** | 실행 연구(Action Research) — 시스템 구축 + 효과 검증 병행 |

---

## 2. 연구 배경 및 필요성

### 2.1 문제 인식

현행 입시 정보 시장은 **광역적·일반적 정보**에 집중되어 있다. 대형 입시 기관이 제공하는 정보는 전국 단위 또는 서울 전역을 대상으로 하기 때문에, 특정 지역 학부모가 실제 의사결정에 필요한 **학교별 세부 특성**을 파악하기 어렵다.

관악구의 경우:
- 일반계 9개교 + 여자고 2개교, 총 11개 고등학교가 밀집
- 학교 간 입시 성과·교육과정·비교과 생태계의 차이가 크지만 체계적으로 비교된 자료 없음
- 학부모가 자녀의 고교 선택·과목 선택·수시 전략 수립 시 근거로 삼을 **지역 특화 데이터**가 부재

### 2.2 연구의 필요성

지역 밀착형 입시정보 시스템은 다음 세 가지 측면에서 필요성을 갖는다.

1. **교육 형평성**: 사교육 접근성이 높은 가정과 그렇지 않은 가정 간 정보 격차를 줄이기 위해, 지역 내 학교 정보를 체계화하여 공유하는 플랫폼이 요구된다.

2. **의사결정 지원**: 초등학교 4학년부터 고등학교 3학년까지 학부모의 진학 의사결정 시점(Critical Moments)마다 적시에 맞춤 정보를 제공하는 체계가 없다.

3. **지속 가능한 컨설팅 기반**: 정량 데이터(입시결과·성적분포)와 정성 데이터(세특 문화·교사 특성·학교 분위기)를 통합한 데이터베이스가 구축될 경우, 근거 기반 컨설팅이 가능해진다.

---

## 3. 연구 목적

본 연구는 다음 세 가지 목적을 가진다.

1. **1차 목적 (시스템 구축)**: 관악구 11개 고등학교에 대한 정량·정성 입시 정보를 수집·정리한 지역 특화 DB를 구축한다.

2. **2차 목적 (서비스 설계)**: 초등~고등 단계별 학부모 컨설팅 워크플로우와 콘텐츠 체계(설명회·뉴스레터·1:1 상담)를 설계하고 운영한다.

3. **3차 목적 (효과 검증)**: 시스템 운영 결과를 추적하여 정보 제공의 실효성과 학부모 의사결정 질 향상 여부를 검증한다.

## 4. 연구 범위 및 대상

### 4.1 공간적 범위
서울특별시 관악구 소재 고등학교 11개교

| 구분 | 학교명 |
|------|--------|
| 일반계 (9개) | 구암고, 당곡고, 남강고, 광신고, 신림고, 삼성고, 인헌고, 성보고, 영락고 |
| 여자고 (2개) | 미림여자고등학교, 문영여자고등학교 |

### 4.2 시간적 범위
- **데이터 수집 기간**: 최근 3~5개년 입시 결과(2021~2025학년도)
- **시스템 운영 기간**: 2026년 5월 ~ 2027년 4월

### 4.3 대상 사용자
- **1차 수혜자**: 관악구 내 초4~고3 학생의 학부모
- **2차 수혜자**: 관련 학생 본인 (특히 고등학생)
- **정보 협력자**: 각 학교 입시담당 교사, 졸업생, 재학생 학부모 커뮤니티

## 5. 연구 내용

### 5.1 데이터 수집 체계 구축

#### 정량 데이터 (필수 수집)
- 최근 3년 수시/정시 진학 비율 및 대학별 진학 현황
- 평가원 모의고사 성적 분포 추이
- 교육과정 편성 현황: 선택과목 조합 및 학생 선택 추이
- 절대/상대평가 비율, 수행평가 배점 구조

#### 정성 데이터 (현장 수집)
- **교사 인터뷰**: 입시담당 교사 대상 — 합격 현황, 세특 기재 관행, 과목 선택 현실
- **학생 인터뷰**: 재학생 및 졸업생 — 학교 분위기, 수업 난이도, 비교과 기회
- **학부모 커뮤니티 분석**: 실제 학부모 반응, 최신 동향

### 5.2 DB 설계 및 구조화

```
관악구_입시정보_DB/
├─ 학교별_상세정보/       ← 11개교 개별 프로파일
├─ 입시결과_통계/         ← 3년 데이터 Excel/Sheet
├─ 시기별_가이드/         ← 고1-3 학년별·시기별 체크리스트
└─ 상담기록/              ← 익명화된 실제 상담 사례
```

### 5.3 컨설팅 서비스 체계 설계

| 대상 | 핵심 의사결정 | 제공 서비스 |
|------|--------------|------------|
| 초4~6 | 중학교 선택 | 강점 진단 + 중학 입시 경로 시뮬레이션 |
| 중1~3 | 고교 준비 방향 | 11개고 분석 설명회 + 목표 고교 상담 |
| 고1 | 수행평가·세특 전략 | 학교별 세특 문화 가이드 + 과목선택 예비상담 |
| 고2 | 과목선택 확정·수시 방향 | 선택과목 최종 가이드 + 세특 적재 전략 |
| 고3 | 원서배치 | 모의고사 기반 입시전략 + 수시/면접 컨설팅 |

### 5.4 콘텐츠 운영 체계

- **분기별 학부모 설명회** (연 4회 이상): 학년별 맞춤 주제
- **월간 뉴스레터**: 입시뉴스 + 11개교 동향 + 학년별 체크리스트
- **학교별 입시데이터 분석 리포트** (분기별): 학교 선택 판단 근거 자료

## 6. 연구 방법

### 6.1 혼합 연구 방법(Mixed Methods)

| 방법 | 내용 | 시기 |
|------|------|------|
| **문헌 분석** | 학교알리미 공시정보, 대학 모집요강, 교육청 자료 분석 | 1~3개월 |
| **심층 인터뷰** | 교사 11명 + 졸업생/재학생 22명 + 학부모 30명 이상 | 2~5개월 |
| **설문 조사** | 설명회 참석 학부모 대상 만족도 및 정보 활용도 조사 | 운영 전 기간 |
| **참여 관찰** | 컨설팅 현장 운영 기록, 상담 사례 축적 | 운영 전 기간 |
| **효과 추적** | 졸업생 입시 결과 추적 (목표대학 진학 여부) | 연간 |

### 6.2 데이터 검증 절차

1. 수집된 정량 데이터는 공식 출처(학교알리미, 평가원)와 교차 검증
2. 정성 데이터는 최소 2개 이상의 독립된 정보원에서 확인(삼각검증)
3. 상충 정보 발견 시 `wiki/` 해당 페이지에 `## 모순·불확실성` 섹션으로 명시

## 7. 기대 성과

### 7.1 직접 성과 (산출물)

| 산출물 | 형태 | 완성 시기 |
|--------|------|----------|
| 관악구 11개교 상세 프로파일 | DB + 마크다운 페이지 | 3개월 이내 |
| 입시결과 통계 자료집 | Excel + 분석 리포트 | 4개월 이내 |
| 학년별 컨설팅 가이드북 | PDF (5종) | 6개월 이내 |
| 월간 뉴스레터 정기 발행 | 디지털 발행물 | 4개월~ 지속 |
| 학부모 설명회 운영 기록 | 자료집 + 참석 데이터 | 4회 이상 |

### 7.2 간접 성과 (효과)

- **정보 접근성 향상**: 관악구 학부모가 학교 선택·과목 선택 의사결정 시 참조할 수 있는 신뢰할 수 있는 자료 제공
- **컨설팅 근거 강화**: 주관이 아닌 데이터 기반 상담 가능
- **지역 교육 네트워크 형성**: 학교 교사·졸업생·학부모 커뮤니티와의 협력 관계 구축

### 7.3 핵심 성과 지표(KPI)

| 지표 | 목표값 |
|------|--------|
| 학부모 설명회 만족도 | 4.5/5.0 이상 |
| 월 신규 상담 신청 | 20명 이상 |
| 재이용율 (형제/지인 소개 포함) | 60% 이상 |
| 목표대학 진학 달성율 | 70% 이상 |
| 뉴스레터 월 조회수 | 500회 이상 |

## 9. 차별화 전략 및 지속 가능성

### 9.1 경쟁 우위 요소

- **지역 독점성**: 관악구 11개 고교를 심층 분석한 DB는 대형 입시기관이 쉽게 복제하기 어려운 자산
- **관계망 기반 정보**: 교사·학생·학부모 커뮤니티를 통해 지속적으로 갱신되는 생생한 정보
- **연속 컨설팅**: 초등~대입까지 한 명의 담당자가 이어주는 일관성 — 장기 고객 관계 형성

### 9.2 지속 가능성 확보 방안

1. **연간 정기 업데이트**: 매년 입시 결과를 반영하여 DB를 갱신 (데이터 陳腐化 방지)
2. **네트워크 확장**: 초기 관악구 → 인접 동작구·금천구로 대상 확장 가능
3. **콘텐츠 수익화**: 무료 기본 정보 + 유료 심화 리포트·1:1 컨설팅 구조로 수익 기반 확보

## 10. 연구 윤리 및 제한점

### 10.1 연구 윤리
- 인터뷰 대상자(교사·학생·학부모)로부터 동의를 취득하고 개인정보를 익명화 처리
- 상담 기록은 본인 동의 없이 외부 공개 불가
- 수집한 학교 정보 중 민감한 사항(예: 특정 교사 실명 부정적 평가)은 DB에서 제외

### 10.2 예상 제한점 및 대응

| 제한점 | 대응 방안 |
|--------|----------|
| 일부 학교의 정보 공개 거부 | 공시 정보 + 학부모 커뮤니티 데이터로 보완 |
| 정성 데이터의 주관성 | 삼각검증(복수 정보원)으로 신뢰도 확보 |
| 입시 정책 변화로 인한 정보 陳腐化 | 연간 업데이트 체계 운영 |
| 시간·인력 제약 | 조교 활용 + 단계별 우선순위 적용 |
