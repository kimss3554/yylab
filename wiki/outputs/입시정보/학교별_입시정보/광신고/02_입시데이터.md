## 🎯 선택과목별 분석

### 2학년 선택과목 개설 현황 (2025학년도 확인)

**수학 영역**
- 수학I (일반선택, 상대평가)
- 기하 (진로선택, 절대평가) — 지필 80%
- ※ 수학II 별도 개설 여부 확인 필요

**과학 영역**
- 물리학I, 화학I, 생명과학I, 지구과학I 모두 개설 (확인됨)

**사회 영역**
- 한국지리, 동아시아사, 경제, 사회·문화, 생활과 윤리 (확인됨)

**3학년 수능 과목 개설 현황 (확인됨)**
- 수학: 미적분, 확률과 통계 (수능 응시과목)
- 과학: 물리학II, 화학II, 생명과학II, 지구과학II
- 사회: 세계지리, 세계사, 정치와 법, 윤리와 사상

### 선택과목 선택 추이

> 실제 학생 선택 데이터는 인터뷰/졸업생 확인 필요

**추정 인기 과목** (개설 과목 구조 기반)
```
수학계열
1순위: 수학I + 미적분 (수능 준비)
2순위: 수학I + 확률과 통계

과학계열 (추정)
자연계: 물리학I + 화학I or 생명과학I
인문계: 생명과학I 단독 선택 (수능 선택 고려)
```
