### 실제 기재 사례 (익명)

**케이스 1: 과학 우수 학생 (대학 자연계 진학)**
```
물리학I 세특:
"수업시간에 직류회로 실험을 통해 옴의 법칙을 이해하고, 
여러 저항값의 조합으로 등가저항을 계산하는 과제에서 
교과서 범위를 벗어나 실제 회로도를 활용한 심화 분석을 
시도하여 높은 평가를 받음. 강사를 통해 접근하지 않고 
자신의 논리로 문제를 해결하려는 적극적인 학습태도 보여줌."

→ 기재 길이: 약 90자
→ 평가: 구체적 + 정량적 + 학생 태도 포함
```

**케이스 2: 사회과학 관심 학생 (대학 인문계 진학)**
```
한국사 세특:
"한국 근현대사 단원에서 3·1운동의 원인과 영향을 분석하는 
과제에서, 국내외 사료를 비교하여 독립운동의 국제적 의의를 
논증한 보고서로 높은 평가를 받음. 역사적 사건을 단순 암기하지 
않고 비판적으로 분석하려는 학습 습관 보여줌."

→ 기재 길이: 약 85자
→ 평가: 역사관 + 탐구능력 + 분석력
```

## 📊 활동 기획 로드맵

### "이 학교 학생이라면 이렇게 활동하라"

**고1**
```
1학기: 동아리 선택 (2-3개 소개 받으며 탐색)
     + 교내 대회 1회 참여 (부담 적은 것)
     + 기초 진로활동 기록

2학기: 동아리 정착
     + 교내 대회 2회 이상 도전
     + 영어/수학 심화 활동
```

**고2 (가장 활동적이어야 할 시기)**
```
1학기: 동아리 리더십 또는 핵심 활동
     + 과학탐구보고서 (여름방학)
     + 교내 대회 3회 이상
     + 학생회 또는 학급임원 겸직

2학기: 동아리 최고 성과
     + 대학 탐방 또는 진로심화
     + 토론대회 참여
```

**고3**
```
초반: 이전 활동 정리 + 기말고사 대비
중반: 대학별고사 준비 중심
```
