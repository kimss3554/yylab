## 🎯 과목별 기재 전략

### 국어

**기재 가능성 높은 활동**
- [ ] 고전 읽고 현대 작품과 비교 분석
- [ ] 추천도서 기반 독서 심화 보고서
- [ ] 수업 발표 (토론, 토의 참여)
- [ ] 창작 글쓰기 (시, 수필, 고백식 에세이)

**기재 사례**
```
"고대 서사시와 현대 소설의 영웅상 변화를 주제로 
<삼국사기>, <삼국유사>, <황금가지>를 연계 읽으며 
문화사적 맥락을 이해하고 2만자의 논문 형식 보고서 작성. 
비판적 텍스트 읽기 능력이 우수함."
```

**준비 체크리스트**
- [ ] 학기초 수업 발표 신청
- [ ] 독서 기록 꾸준히 제출
- [ ] 보고서 품질 높이기
- [ ] 교사와 상담 (활동 기재 신청)

---

### 수학

**기재 가능성 높은 활동**
- [ ] 심화 탐구과제 (고난도 문제 풀이)
- [ ] 수학적 모델링 (실생활 문제 수식화)
- [ ] 증명 기반 활동
- [ ] 프로젝트 수행

**기재 사례**
```
"2차 함수와 원의 위치 관계를 이용하여 현수막 
구조의 수학적 원리를 분석하고 설계도를 작성하는 
프로젝트에서 기하학적 직관력과 수식 표현 능력이 
뛰어남."
```

**준비 체크리스트**
- [ ] 탐구과제 충분히 준비하기
- [ ] 수학적 과정 기록하기 (풀이과정 메모)
- [ ] 교사 면담 (난이도 높은 문제 상담)
- [ ] 조별과제시 주도적 역할

---

### 영어

**기재 가능성 높은 활동**
- [ ] 영어 발표 (프레젠테이션)
- [ ] 영어 작문 (에세이, 일기)
- [ ] 영어 번역 기반 활동
- [ ] 영어 토론

**기재 사례**
```
"현대 영문학의 주제를 분석하는 수업에서 
<The Great Gatsby>를 정독하고 비평문을 작성하여 
영어 읽기 이해력과 비판적 사고가 우수함을 보임."
```

**준비 체크리스트**
- [ ] 수업 발표 참여
- [ ] 과제물 품질 높이기
- [ ] 원어민 표현 공부
- [ ] 교사 피드백 반영

---

### 과학 (물리/화학/생과/지과)

**기재 가능성 높은 활동**
- [ ] 탐구보고서 (실험 기반)
- [ ] 과학 논문 리뷰
- [ ] 심화 스터디 (고난도 개념 학습)
- [ ] 과학적 문제해결

**기재 사례**

**물리학I:**
```
"전기장의 개념을 이해하기 위해 디지털 기기를 활용하여 
정전유도 현상을 직접 관찰하고, 이론과 실험 결과를 
비교 분석하는 심화 학습을 주도적으로 수행하여 물리적 
직관력이 뛰어남."
```

**화학I:**
```
"산화-환원 반응의 원리를 실생활 사례(배터리 원리, 
부식 방지)에 적용하여 보고서를 작성하고, 
화학 반응식 계산에서 일관되게 높은 수준을 보임."
```

**생명과학I:**
```
"유전 현상과 진화 이론을 연계하여 이해하고, 
멘델의 법칙을 현대 생명공학 사례에 적용하는 
심화 과제를 수행하여 생명과학적 사고능력이 우수함."
```

**준비 체크리스트**
- [ ] 실험 보고서 철저히 작성
- [ ] 고난도 개념 정리하기
- [ ] 교사와 질문 상담 (그 과정 기재 가능)
- [ ] 수행평가 최상위 결과 목표

---

### 사회과학 (한국사/지리/경제/윤리)

**기재 가능성 높은 활동**
- [ ] 사료 분석 기반 보고서
- [ ] 사회 현상 분석 및 논평
- [ ] 역사 에세이
- [ ] 정책 제안 과제

**기재 사례**

**한국사:**
```
"일제 강점기 독립운동을 국내 사료와 해외 기록을 
통해 비교 분석하는 세미나를 주도적으로 진행하여 
역사적 통찰력과 비판적 분석능력이 우수함."
```

**경제:**
```
"금리 인상이 부동산 시장에 미치는 영향을 분석하는 
프로젝트에서 경제 데이터를 수집, 분석하고 
실증적 근거에 기반한 보고서를 작성하여 
경제 분석능력이 뛰어남."
```

**준비 체크리스트**
- [ ] 주제별 깊이 있는 탐구보고서
- [ ] 신문/기사 분석하기
- [ ] 토론 활발히 참여
- [ ] 시사 관심도 높이기

## 🔄 세특 기재 프로세스

### 학기 초 (3월/9월)
```
Step 1: 이번 학기 목표 활동 계획
- "어떤 활동을 통해 세특을 받을 것인가?"
- "어떤 과목에 집중할 것인가?"

Step 2: 교사 면담 (개학 초)
- "이번 학기 어떤 활동이 도움될까요?"
- "세특 기재 가능한 활동을 알려주세요"

Step 3: 활동 시작
- 수행평가 최고 점수 목표
- 추가 과제 주도적으로 신청
```

### 학기 중
```
Step 1: 활동 기록 (메모, 사진, 인증)
- 구체적 내용 남기기
- 날짜/시간 기록

Step 2: 교사와의 상호작용
- 질문하기 (교사가 학생 역량을 알도록)
- 결과 공유하기

Step 3: 추가 학습 신청
- 방과후 학습 신청
- 심화 과제 신청
```

### 학기 말 (6월/12월)
```
Step 1: 교사 면담 신청
- "제가 한 활동을 정리해봤는데..."
- 한두 가지 핵심 활동 어필

Step 2: 기재 시기 모니터링
- 생기부 공시일 전후로 확인
```
