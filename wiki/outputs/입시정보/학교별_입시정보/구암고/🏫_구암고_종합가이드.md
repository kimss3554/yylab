## 🎯 학년별 준비 로드맵

### 고1: 기초 다지기

**1학기 (3월~6월)**
```
📌 초점: 첫 학기 성공 + 2학년 준비
- 첫 수행평가 차근차근 대비
- 동아리 2-3개 선택
- 교내 대회 1회 참여 경험
- 진로활동 활발히 참여

✅ 체크리스트:
- 수행평가 전 교사 상담 O
- 수업 적극 참여 O
- 모둠과제에서 역할 수행 O
- 개학초 성적 평가 후 목표 재설정 O
```

**2학기 (7월~2월)**
```
📌 초점: 세특 기재 경험 + 과목선택 탐색
- 수행평가에서 산출물 제출 (보고서 등)
- 동아리 정착
- 교내 대회 2회 이상
- 진로활동 기록 충실

✅ 체크리스트:
- 탐구보고서 최소 1회 제출 O
- 과목별 심화활동 1회 이상 O
- 겨울방학 선택과목 탐색 O
- 진로활동 자료 정리 O
```

**고1 준비물**
- 📕 주교재 및 보충자료
- 📔 과목별 노트 (수행평가 대비용)
- 📊 수행평가 결과 기록장
- 📅 활동 일정 캘린더

---

### 고2: 가장 중요한 해

**1학기 (3월~6월)**
```
📌 초점: 과목선택 확정 + 세특 본격화
- **4월: 과목선택 최종 결정** ⭐
  └─ "구암고라면 어떤 조합이 유리한가?" 분석 필요
- 수행평가/탐구보고서 고품질
- 교내 대회 3회 이상
- 학생회/학급임원 또는 동아리 리더

✅ 체크리스트:
- 선택과목 최종 결정 O
- 수행평가 상위 3% 목표 O
- 탐구보고서 2회 이상 제출 O
- 세특 기재 받기 위한 활동 신청 O
```

**2학기 (7월~2월)**
```
📌 초점: 세특 극대화 + 진로심화
- 여름방학 탐구보고서 (여름과제)
- 동아리 최고 성과 도출
- 토론대회 참여
- 대학 탐방 (진로활동)
- 대입 정시/수시 기초 분석 시작

✅ 체크리스트:
- 여름방학 탐구보고서 완성 O
- 동아리에서 리더십 발휘 O
- 교내 대회 2회 이상 입상 O
- 생기부 공시 후 세특 검토 O
```

**고2 준비물**
- 📊 선택과목 비교 분석표
- 📝 세특 기재 사례 모음
- 🔬 탐구보고서 템플릿
- 📚 대학 입시 정보 수집

### 고3: 입시 최종 결정

**1월~6월 (상반기)**
```
📌 초점: 수시/정시 선택 + 원서 준비
- 3월/4월/6월 모평 분석
  └─ "내 성적이라면 수시? 정시?"
- 지난 세특 검토 (추가 보완 기회 찾기)
- 대학 모집요강 분석 시작
- 수시 목표 대학 리스트업

✅ 체크리스트:
- 3월 모평 후 목표대학 설정 O
- 4월 모평 후 과목선택 적정성 판단 O
- 6월 모평 후 수시/정시 비중 결정 O
- 수시 6개 후보 대학 선정 O
```

**7월~8월 (수시 준비)**
```
📌 초점: 수시 원서 최종 결정
- 수시 6개 대학 최종 결정
- 대학별 자소서 요구사항 확인
- 자소서 및 면접 준비 시작
- 정시 수능 기초 다지기

✅ 체크리스트:
- 수시 6개 배치 완료 O
- 자소서 주제 선정 O
- 모의면접 1회 이상 O
- 정시 선택과목 최종 확인 O
```

**9월~10월 (수시 최종)**
```
📌 초점: 수시 시험 집중 + 정시 준비
- 대학별고사 준비
- 면접 연습 (학교별 특성 파악)
- 9월 모평 분석 (정시 목표선정)
- 수능 100일 집중

✅ 체크리스트:
- 대학별 고사 기출 분석 O
- 모의면접 2-3회 완료 O
- 9월 모평 성적 분석 O
- 정시 목표점수 설정 O
```

**11월 (수능 후)**
```
📌 초점: 최종 입시전략 수립
- 수능 성적 분석
- 수시 합격 예측
- 정시 지원 전략 수립
- 대학 선택 최종 회의

✅ 체크리스트:
- 수능 성적 분석 O
- 수시 합격 확률 계산 O
- 정시 지원 대학 순서 결정 O
- 최종 선택 결정 O
```
