## 🎯 시스템 소개

이 시스템은 **관악구 중심의 맞춤형 입시정보 플랫폼**입니다.
초등학교부터 대학입시까지 **전 단계적 진학 로드맵**을 제공하며,
**지역 특화 정보** + **학생별 맞춤 컨설팅**을 통해 입시 성공을 돕습니다.

### 3️⃣ 학생별 1:1 컨설팅을 진행 중이라면?

**학생 추가하기**
```
1. 학생_생기부_컨설팅_기록/ 폴더로 이동
2. 새로운 폴더 생성: [학생명]/
3. 기본정보 파일 작성: [학생명]_기본정보.md
```

**학기별 기록 관리**
- 매 학기마다 [[학생_생기부_컨설팅_기록/📝_학생별_기록_가이드|기록 템플릿]] 사용
- 수행평가 결과 입력
- 탐구보고서 현황 기록
- 세특 기재 현황 업데이트

**입시 전략 수립**
- 고3 초반 [[학생_생기부_컨설팅_기록/📝_학생별_기록_가이드|입시전략_로드맵]] 작성
- 수시/정시 선택 → 원서 배치 → 월별 계획

### 🏫 학교별_입시정보/

**대상**: 고등학교 선택을 고민하는 중학 학부모 / 고등학생

**포함 내용**
```
✅ 구암고 (완성) - 템플릿 모델
✅ 당곡고 ~ 문영여자 (10개) - 준비 중
   → 각 학교마다 6개 파일
      1) 종합가이드 (인덱스)
      2) 기본정보 (학교 개요)
      3) 입시데이터 (3년 입시결과)
      4) 교육과정 (선택과목, 수행평가)
      5) 비교과활동 (동아리, 세특)
      6) 세특기재문화 (과목별 기재 전략)
```

**활용법**
```
중학 학부모:
"우리 아이가 구암고 갈 수 있을까?"
→ [[학교별_입시정보/구암고/02_입시데이터|입시데이터]]로 합격선 확인
→ [[학교별_입시정보/구암고/04_비교과활동|비교과활동]]로 학교문화 파악

고1 학생:
"첫 수행평가 어떻게 준비할까?"
→ [[학교별_입시정보/구암고/03_교육과정|교육과정]]에서 과목별 평가방식 확인
→ [[학교별_입시정보/구암고/05_세특기재문화|세특기재문화]]에서 팁 학습

고2 학생:
"과목선택 어떻게 할까?"
→ [[학교별_입시정보/구암고/02_입시데이터|입시데이터]]에서 선배들 선택 조합 분석
→ [[학교별_입시정보/구암고/03_교육과정|교육과정]]에서 난이도 비교
```

### 📄 상담기록_템플릿

**대상**: 모든 상담 기록

**포함 내용**
```
✅ 기본정보 입력 (학생명, 학교, 학년)
✅ 학생 현황 분석 (성적, 강점, 약점)
✅ 상담내용 기록 (주제, 질문, 우리의 조언)
✅ 액션플랜 (즉시과제, 학기목표, 추천사항)
✅ 학생 기록 (교과, 비교과, 세특)
✅ 진행과정 추적 (상담히스토리, 성과검토)
```

**활용법**
```
학부모 상담 후:
→ 상담기록_템플릿 사용
→ 상담내용 기록
→ 액션플랜 제시

학기 중:
→ 학생 활동 기록 업데이트
→ 진행상황 추적

학기말:
→ 성과 검토
→ 다음 학기 전략 수정
```

**Q1. 구암고 정보만 상세하고 다른 학교는 왜 간단하나?**
```
A. 구암고는 템플릿 모델입니다.

```
A. 각 특목고를 방문한 후 정보를 수집하여 
   개별 파일로 만들 예정입니다.
   (예상: 2024년 6월-8월)
```

```
A. 네. Obsidian 로컬 저장소이므로

   학생/학부모 동의 하에만 작성하시고,
   필요시 이름을 가명으로 기록하셔도 됩니다.
