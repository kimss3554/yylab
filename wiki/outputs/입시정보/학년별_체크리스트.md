# ✅ 학년별 월간 체크리스트

## 📚 초등학교 (4-6학년)

### 🎯 초등 4학년

#### 연간 목표
```
- 기초학력 완성 (읽기, 쓰기, 계산)
- 학습습관 형성
- 자신의 특기 발굴
```

#### 월별 체크리스트

**1월 (신학기 준비)**
- [ ] 새학기 교재 미리보기
- [ ] 학용품 정리
- [ ] 겨울방학 학습 마무리
- [ ] 방학 독서 기록 정리

**2월 (입학 준비)**
- [ ] 새 학교/새 담임선생님 정보 파악
- [ ] 시간표 확인 및 준비물 체크
- [ ] 학교 가는 길 미리 가보기
- [ ] 신입생 오리엔테이션 참석

**3월 (학교 적응)**
- [ ] 학교생활 적응 확인
- [ ] 담임선생님과 대화
- [ ] 새 친구들과 관계 형성
- [ ] 첫 평가 준비

**4월 (기초학습 강화)**
- [ ] 2주 1회 학습상태 확인
- [ ] 독서 습관 점검
- [ ] 계산 능력 평가
- [ ] 특기 탐색 프로그램 참여 (선택)

**5월 (학교 문화 적응)**
- [ ] 반 친구들 관계 점검
- [ ] 학교 행사(체육대회 등) 참여
- [ ] 수업 이해도 확인
- [ ] 개인상담(담임) 신청

**6월 (1학기 말)**
- [ ] 1학기 학습 성과 검토
- [ ] 통知簿 확인 및 해석
- [ ] 보충 필요 영역 파악
- [ ] 여름방학 계획 수립

**7월 (여름방학)**
- [ ] 부족한 영역 보충학습
- [ ] 독서 프로그램 참여
- [ ] 문해력 향상 활동
- [ ] 실내 야외 활동 균형

**8월 (여름방학 말기)**
- [ ] 2학기 준비 (물품 구입)
- [ ] 방학 학습 성과 정리
- [ ] 새학기 심리 준비
- [ ] 개학 전 일상 복귀

**9월 (2학기 시작)**
- [ ] 새학기 적응
- [ ] 2학기 목표 설정
- [ ] 반 임원 선출 참여 (선택)
- [ ] 가을 학교 행사 확인

**10월 (체험학습 시즌)**
- [ ] 추계 현장학습 준비
- [ ] 가족 문화생활 활동
- [ ] 계절 독서 활동
- [ ] 과학 관찰 (식물, 곤충 등)

**11월 (학습 점검)**
- [ ] 2학기 중간 성적 확인
- [ ] 보충학습 필요 여부 판단
- [ ] 특기 발전 현황 점검
- [ ] 겨울방학 준비

**12월 (2학기 말 & 연말)**
- [ ] 2학기 학습 성과 종합 평가
- [ ] 통知簿 최종 확인
- [ ] 연 1회 상담 (생기부 전혀 없으니 부모상담)
- [ ] 겨울방학 학습 계획 수립
