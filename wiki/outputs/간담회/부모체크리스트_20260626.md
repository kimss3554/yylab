# 고등학교 선택 전 확인 사항

> 자녀와 함께 읽으며 체크해 보세요. 체크가 많이 될수록 결정이 쉬워집니다.

---

## ✅ 학교 선택 체크리스트

### 학교 환경
- [ ] 남자고 / 여자고 / 공학 중 아이가 선호하는 환경을 물어봤다
- [ ] 학교 규모(학년당 인원)가 아이 성향과 맞는지 생각해봤다 *(소형 vs 대형)*
- [ ] 기독교 계열 학교(영락·문영여고)에 대한 거부감 여부를 확인했다
- [ ] 통학 거리와 교통 편의를 확인했다

### 학업 환경
- [ ] 관심 학교의 4년제 진학률을 확인했다
- [ ] 관심 학교의 수행평가·세특 환경이 아이에게 맞는지 생각해봤다 *(서술형 강함 vs 발표 강함)*
- [ ] 아이의 강점 교과(국어/수학/영어/과학/사회)가 무엇인지 파악했다
- [ ] 진로 계열(인문·이공·예술 등)이 대략 정해졌거나, 정할 계획이 있다

### 직접 확인
- [ ] 학교 홈페이지에서 교육과정 편성표를 한 번이라도 봤다
- [ ] 후보 학교 2~3곳을 실제로 방문할 계획이 있다

## 📚 여름방학 준비 타임라인

| 시기 | 할 일 |
|------|------|
| **6월 말** | 학교 후보 2~3개로 압축 / 도서 1권 시작 |
| **7월** | 도서 2~3권 완독 / 관심 탐구 주제 1개 정하기 / 학교 방문 |
| **8월** | 간단한 탐구 기록 써보기 / 수학 기초 점검 / 고1 수업 준비 |
