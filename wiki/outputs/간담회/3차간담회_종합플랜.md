# 생각탐구 여름 전환 프로그램 — 3단계 유입형 간담회

> **핵심 원칙:** 간담회는 정보 제공이 아니라 **다음 상품으로 가는 다리**다.  
> 간담회가 끝났을 때 바로 들어갈 상품이 준비되어 있어야 한다.

## 전략 프레임

### 타깃 재설계 (초등 저학년 제외)

| 차수 | 대상 | 즉시성 | 전환 상품 |
|------|------|-------|---------|
| **1차** | 중3·예비고1 학부모 | 당장성 (여름이 마지막 기회) | 중3 필수특강 · 고교선택 1:1 컨설팅 |
| **2차** | 중1·중2 학부모 | 필요성 (지금 안 하면 늦는다) | 중등 주제탐구 정규반 · 탐구보고서 특강 |
| **3차** | 초4~6 학부모 | 장기 유입 (정규 수업 기반 고객) | 초등 고학년 독서기반 탐구글쓰기반 |

> **초등 저학년은 이번 여름 제외.** 전환율이 낮고 장기 고객이라 지금 당장의 매출과 거리가 멀다.  
> 브랜드가 안정된 뒤 "골든타임 독서 워크숍"으로 별도 기획.

### 이번 여름 홍보 핵심 메시지 (전 채널 통일)

> **"고등학교 가서 필요한 수행평가·탐구보고서·서논술형 역량을  
> 중학교와 초등 고학년부터 준비시키는 곳"**

한 줄:
> **생각탐구는 독서와 글쓰기를 수행평가, 탐구보고서, 생기부 방향까지 연결합니다.**

## 공통 전환 장치: 우리 아이 고교 준비 진단 쿠폰

모든 차수에 공통으로 적용. **검사 자체가 아니라 해석과 코칭이 핵심.**

### 쿠폰 구성

| 항목 | 내용 |
|------|------|
| 학생 다중지능검사 | 학습 강점·탐구 방향 파악 |
| 학생 성격유형검사 | 경쟁·협업·표현 성향 |
| 학부모 성격유형검사 | 부모의 기대 방식 |
| 학생·학부모 결과 비교 해석 | 인식 차이 발견 (전환 핵심 포인트) |
| 30분 해석 코칭 | 고교 선택·탐구 방향 맞춤 정리 |

### 왜 강력한가

> "부모는 아이를 안다고 생각하지만, 실제로는 부모의 기준으로 아이를 판단하는 경우가 많습니다.  
> 검사 결과를 비교해 보여주면 학부모가 바로 느낍니다 — '아, 내가 아이를 내 기준으로 보고 있었구나.'"

이 순간이 **상담 전환 포인트**다.

### 운영 원칙

- **완전 무료 X** → 진지도 하락. **검사비 무료 + 해석 코칭 우선 예약권** 방식
- 사용 기한: **간담회 후 7일 이내 예약** (기한 없으면 안 씀)
- 대상: 간담회 참석 가정 한정

### 간담회에서 말할 때

> "오늘 학교 이야기를 드렸지만, 학교 선택 전에 더 중요한 게 있습니다. 아이를 먼저 봐야 합니다.  
> 부모님이 보는 아이와 실제 아이의 성향이 다를 때가 많아요.  
> 오늘 참석하신 분들께 학생·학부모 성향 비교 진단 쿠폰을 드립니다.  
> 7일 이내 예약하시면 결과 해석과 방향 코칭을 함께 드립니다."

### 쿠폰 이후 분기

| 검사에서 드러난 문제 | 연결 상품 |
|----------------|---------|
| 고교 선택 고민 큼 | 고교 선택 1:1 컨설팅 |
| 글쓰기·탐구 경험 부족 | 중3 필수특강 · 중등 주제탐구반 |
| 진로 방향 불명확 | 진로·탐구 방향 컨설팅 |
| 부모 기대 ≠ 아이 성향 차이 큼 | 학부모 상담 + 학생 수업 병행 |

## 1차 간담회 — 중3·예비고1 학부모

### 핵심 정보

| 항목 | 내용 |
|------|------|

| 대상 | 중3 (예비고1 포함) 학부모 |
| 즉시 연결 상품 | 중3 고교입학 전 필수특강 · 고교선택 1:1 컨설팅 |
| 예정일 | 2026-06-26 (금) 11:00 |

### 학부모 핵심 고민

- "어느 학교가 내 아이에게 맞을까?"
- "내신이 유리한 학교가 있을까?"
- "관악구 고교 분위기가 어떻게 다른가?"
- "고등학교 가서 무너지지 않으려면 뭘 해야 하나?"
- "독서·수행평가·탐구보고서를 뭘 준비해야 하나?"

### 90분 구성

| 시간 | 섹션 | 내용 | 목적 |
|------|------|------|------|
| 0–10분 | 오프닝 | 학부모 고민 라운드 (1인 30초) → 패턴 묶기 | 공감 형성 |
| 10–35분 | 고교 선택 기준 | 관악구 3유형 분류 + 부모 기준 vs 실제 기준 비교 + 기준 전환 | 인식 전환 |
| 35–55분 | 수행·탐구 적응력 | 수행평가 40~60% 현실 + 탐구보고서 Before/After 실물 | 신뢰·증거 |
| 55–70분 | 중3 방학 로드맵 | 7월 기말 후~고1 입학까지 시기별 준비 | 특강 필요성 |
| 70–83분 | 체크리스트 + 쿠폰 | 우리 아이 고교 선택 체크리스트 작성 → 진단 쿠폰 소개 | 전환 장치 |
| 83–90분 | 신청서 + 미니상담 | 상담 예약 카드 작성 → 1:1 대화 | 실질 전환 |

### 1차 핵심 멘트 / 데이터 (바나나 출처)

> **[줄탁동시 오프닝]**  
> "병아리가 알에서 나올 때, 안에서 쪼고 밖에서 쪼는 게 동시에 이루어져야 합니다.  
> 부모가 아무리 두드려도 아이 안에서 준비가 안 되면 못 나와요.  
> 그리고 반대로, 아이가 준비됐는데 부모가 타이밍을 못 잡으면 기회를 놓칩니다.  
> 지금이 바로 그 타이밍입니다."

> **[65.1% 데이터 — 고교 선택 파트 말미]**  
> "입시전문기업 데이터를 보면, 전체 학생의 65.1%가 고1 1학기 등급을 고3까지 유지합니다.  
> 반대로 말하면, 지금 이 순간의 준비가 곧 대입 결과입니다."

> **[절대→상대 착시]**  
> "중학교 A등급은 상위 10~40%예요. 고등학교 1등급은 상위 10%입니다.  
> 중학교 때 A 받던 아이가 고교 첫 시험에서 3등급 받는 경우가 적지 않습니다."

> **[수행평가 현실]**  
> "관악구 대부분 고등학교에서 수행평가가 40% 내외입니다.  
> 논술·탐구보고서·발표. 이게 다 쓰는 힘에서 나옵니다."

> **[서울대 2028 — 마지막 핵심 한 방]**  
> "서울대가 2028학년도부터 정시에서도 교과역량평가 40%를 봅니다.  
> 수능 점수만으로 가는 시대가 끝나가고 있어요."

### 1차 방학 로드맵 (월별 체크사항 출처)

> 고1 3~4월이 한 해를 결정합니다. 동아리·수행평가 유형 파악·탐구주제 선정이 모두 이 두 달에 집중됩니다.  
> 여름방학(8월)은 1학기 내신 기반으로 진로를 점검하고 2학기 전략을 세우는 전환점입니다.

**배포 자료로 연결:** [[sources/월별체크사항_고1]] — 고1 3월~고2 2월 흐름 한눈에 보여주는 표

### 제공 자료 (2종)

1. **관악구 고교 분석자료** — 3유형 분류·수행평가 유형 진단·기준 비교표 (A4 1장)
2. **중3 기초 진단쿠폰** — 7/11 3차에서 결과 해석 (A5 카드)

> Before/After 실물·로드맵·체크리스트는 현장 시연. 도서 리스트·달력은 3차(7/11) 배포.

## 연결 상품 1-A: 중3 고교입학 전 필수특강

### 포지셔닝

> "고등학교 가서 처음 탐구보고서를 쓰는 아이와, 중3 때 한 번이라도 구조를 익혀본 아이는 시작점이 다릅니다."

|------|------|------|

| 4회 | 탐구보고서 완성 + 발표 + 자기평가 | 최종 보고서 + 발표 자료 |

4회 필수특강 → 결과물 공유 → 중등 정규반 or 고1 컨설팅 전환

## 3차 간담회 — 초등 4~6학년 학부모

### 핵심 정보

| 항목 | 내용 |

| 예정일 | 2026년 7월 말~8월 초 |

### 학부모 핵심 고민

- "책은 읽는데 독서감상문을 못 쓴다"
- "글이 너무 짧고 자기 생각이 없다"
- "발표를 싫어한다"
- "중학교 가기 전에 뭘 준비해야 하나?"
- "논술학원과 뭐가 다른가?"

### 90분 구성

| 시간 | 섹션 | 내용 | 목적 |
|------|------|------|------|
| 0–10분 | 오프닝 | 학부모 고민 라운드 → 패턴 묶기 | 공감 |
| 10–30분 | 책과 글 사이의 빈틈 | "읽기"와 "쓰기" 사이에 빠진 것: 질문과 생각 | 문제 인식 |
| 30–50분 | 생각이 자라는 독서법 | 줄거리 요약 → 질문 독서 → 탐구 연결 3단계 | 방법론 소개 |
| 50–65분 | 3문장 글쓰기 → 탐구글쓰기 | 나는 ~다 / 왜냐하면 / 예를 들어 → 의견글·서평·탐구글 | 실천 가능성 |
| 65–80분 | 글쓰기 상태 진단 | 6항목 체크리스트 → 현재 수준 파악 | 필요성 인식 |
| 80–87분 | 독서·글쓰기 진단 쿠폰 | 학생 발달 수준 + 학부모 기대치 비교 | 전환 장치 |
| 87–90분 | 신청서 + 미니상담 | | 실질 전환 |
