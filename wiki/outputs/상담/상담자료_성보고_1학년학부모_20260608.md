# 성보고 1학년 학부모 상담 자료

> **상담 전 체크**: 학생 현재 내신 등급, 희망 진로·전공, 수학 수준, 문이과 방향 파악 필요

## 2. 성보고만의 강점 — 관악구 내 유일한 특징

| 특징 | 내용 | 의미 |
|------|------|------|
| **논술 필수 (3학점)** | 교양과목 전원 이수 | 관악구 11개교 중 **유일** |
| **철학 필수 (5학점)** | 교양과목 전원 이수 | 관악구 11개교 중 **유일** |
| **미적분Ⅰ 필수** | 수학 학교지정 | 광신고·미림여고·인헌고와 공통 |
| **영어 6과목** | 공통2+영Ⅰ·Ⅱ+독해작문+영어권문화 | 타교 대비 많음 |
| **창의경영** | 3학년 선택 | 경영·경제 진로 독특 과목 |
| **세계문제와미래사회** | 사회교과 학교지정 | 글로벌 이슈 교육 |

> **논술·철학 필수는 성보고의 가장 큰 차별점입니다.** 대입 논술전형을 목표로 하거나, 인문학적 사고 능력을 쌓고 싶은 학생에게 특히 유리한 환경입니다.

## 3. 성취도로 본 성보고 학생 분포

*2023학년도 1학년 기준*

| 과목 | 성보고 평균 | A비율 | E비율 | 해석 |
|------|-----------|------|------|------|
| 국어 | **79.2점** | 25.2% | 8.4% | 관악구 상위 |
| 수학 | 72.8점 | 10.5% | 16.8% | 평균은 높지만 A 적음 |
| 영어 | **76.0점** | 28.2% | 19.7% | 영어 E비율 주의 |
| 통합사회 | **79.3점** | 23.5% | 4.6% | 관악구 최고 수준 |
| 통합과학 | 71.7점 | 14.3% | 24.4% | 과학 기초 격차 |

**수학 해석 포인트:**
수학 평균 72.8점은 관악구 내 가장 높은 수준이지만, A비율(10.5%)이 낮습니다. **중간층이 두텁고 최상위층이 적은 분포**입니다. 즉, 수학을 극상위로 가져가고 싶은 학생에게는 내신 경쟁이 다소 유리할 수 있습니다.

## 4. 상담 핵심 — 성보고의 3가지 시나리오

### 시나리오 A: 성보고 장점 극대화 (인문·논술 방향)

**해당 학생:** 국어·영어 강세, 인문사회계열 진로, 논술전형 관심

- 논술·철학 필수 이수로 자연스럽게 논술전형 준비 가능
- 국어(79.2점)·사회(79.3점) 높은 학교 평균 → 인문계 내신 유리 가능
- 영어 6과목 이수로 영어 실력 향상 기회
- **세특 전략:** 논술·철학 수업에서 진로 연계 세특 작성 (타교 학생이 쓸 수 없는 고유 이력)
- **추천 선택과목:** 사회·문화, 생활과윤리, 창의경영, 경제수학

### 시나리오 B: 이공계 진입 (수학·과학 방향)

**해당 학생:** 수학 보통 이상, 이공계 진로 희망

- 미적분Ⅰ 필수 이수로 기초 다짐
- 2~3학년 물리Ⅱ·화학Ⅱ·생명과학Ⅱ·지구과학Ⅱ 4개 모두 개설 → 탐구 이수 가능
- 수학 A비율(10.5%)이 낮아 **최상위 수학 내신 도전 여지 있음**
- **주의:** 이공계 심화과목 다양성은 영락고(고급수학·공학일반)나 광신고보다 적음

### 시나리오 C: 내신 관리 집중 (안정적 내신 목표)

**해당 학생:** 현재 내신 3~4등급, 내신관리형 수시 목표

- 관악구 4위 진학률 학교 — 전반적 학업 분위기 안정
- 학업중단율 1.1% (관악구 평균 2.1% 대비 절반) → 안정적 학교 환경
- 신입생 감소(169명)로 **학급 수 축소 → 내신 산출 단위가 줄어 유불리 변동 가능** → 확인 필요

## 5. 타교와의 핵심 비교 포인트

### 성보고 vs 영락고 (진학률 유사, 53.9%)

| 항목 | 성보고 | 영락고 |
|------|-------|-------|
| 진학률 | 51.0% | 53.9% |
| 유형 | 남자고 | 남녀공학 |
| 특징 | 논술·철학 필수 | 고급수학·공학일반·정보 체계 |
| 수학 E비율 | 16.8% | 26.6% |
| 영어 A비율 | 28.2% | 48.2% |
| 특이사항 | 인문학 강화 | 기독교 계열, SW·IT 특화 |

→ 이공계·IT 진로라면 영락고, 인문·논술이라면 성보고가 더 유리한 환경

### 성보고 vs 인헌고 (상승세 학교)

| 항목 | 성보고 | 인헌고 |
|------|-------|-------|
| 진학률 | 51.0% (안정) | 47.7% (↑ 급상승) |
| 수학 평균 | 72.8점 | 55.3점 |
| 영어 A비율 | 28.2% | 49.7% (관악구 1위) |
| 특색 | 논술·철학 | 국제교류·예술 4종 필수 |

→ 영어 강점 + 국제 진로라면 인헌고, 수학 기초를 탄탄히 쌓으려면 성보고

### 성보고 vs 광신고 (진학률 1위)

| 항목 | 성보고 | 광신고 |
|------|-------|-------|
| 진학률 | 51.0% | 58.0% |
| 규모 | 609명 | 486명 |
| 학급 방식 | 남자고 | 남녀분리학급 |
| 국어 A비율 | 25.2% | 34.4% |
| 통합과학 E비율 | 24.4% | 14.8% |

→ 성보고는 이미 진학률 51%로 준상위권 — 광신고로 옮길 수 없는 상황이므로, **현 학교에서 어떻게 전략을 세울지**가 핵심

## 6. 현 1학년 부모 상담 — 지금 당장 확인해야 할 것

### 즉시 확인 사항 (1학년 1학기)

1. **현재 중간고사 성적 파악** — 과목별 원점수, 반 평균, 표준점수 확인
2. **수학 위치 확인** — 학교 수학 평균 72.8점 기준으로 상/중/하 어디인지
3. **논술·철학 수업 태도** — 부담으로 느끼는지 vs 흥미로운지 (진로 방향 힌트)
4. **2학년 선택과목 사전 정보** — 기하 선택 여부(이공계 여부 결정적)
5. **세특 준비 시작 여부** — 1학년부터 누적이 중요

### 학부모가 오해하기 쉬운 것

- "성보고 4위니까 나쁘지 않죠?" → **진학률은 평균이고, 내 아이 위치를 봐야 함**
- "논술·철학 왜 억지로 들어요?" → **논술전형 비중 대학별 확인 후 전략화 가능**
- "신입생 줄어드는 거 괜찮나요?" → 1학년 169명 → 내신 산출 인원·학급 수 변동 영향 체크 필요

## 7. 상담 시 추가로 물어볼 것

- 자녀의 **국어 vs 수학** 비교 강점
- **논술전형** 고려 여부 (성보고 논술 필수 수업이 직접 도움)
- 이공계·인문계 **방향 결정 여부**
- **재수 고려 여부** — 남자고 특성상 재수 비율이 높을 수 있음 (진학 합계 68.7%로 나머지 31%는 기타)
- **학원·사교육 현황** — 현재 학교 수업과의 조율 필요
