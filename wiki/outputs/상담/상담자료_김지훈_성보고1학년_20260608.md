## 5. 진학 전략

### 5-1. 핵심 방향

| 항목 | 판단 |
|------|------|
| 전형 | **수시 학종 집중** — 수능 최저 없는 전형 타겟 |
| 논술전형 | **배제** — 국어·영어 약세, 수행평가 기록 낮음 |
| 교과전형 | 보조 옵션 — 내신 흐름 보며 결정 |
| 목표 전공 | AI/SW/컴퓨터공학 계열 |
| 수능 병행 | 학종 우선, 수능은 3학년 이후 병행 검토 |

### 5-2. 학종 스토리 완성도

| 구성 요소 | 현황 | 완성도 |
|---------|------|------|
| IT 동아리(프로토타입) | 있음, 내용 미기재 | 🔄 |

```
2학년: 인공지능기초 + 데이터과학
3학년: 프로그래밍 (공동) + 융합과학탐구
         ↓
  SW 개발 → AI 이론 → 데이터 분석 → 코딩 심화
```

## 10. 관악구 공동교육과정 안내

*⚠️ 2026년 실제 개설 과목·일정은 학기 초 성보고 교육과정 담당 선생님 또는 서울시교육청 공동교육과정 포털(jccu.eduinfo.kr)에서 확인 필요*

---

### 공동교육과정이란?

학교 자체에서 개설하지 못한 과목을 인근 학교, 온라인, 대학 등과 연계해 이수하는 제도. 이수 시 **학생부 교과학습발달상황에 정식 기재**되며 학점으로 인정됨.

---

### 운영 유형 3가지

| 유형 | 방식 | 장점 | 단점 |
|------|------|------|------|
| **거점학교형** | 방과 후 인근 학교로 이동해 수업 | 대면 수업, 실험·실습 가능 | 이동 시간 부담 |
| **온라인형** | 실시간 화상수업 | 이동 없이 수강 가능 | 집중도 관리 필요 |
| **대학 연계형** | 대학 캠퍼스에서 수강 | 대학 시설·교수진 활용 | 이동 거리, 선발 경쟁 |

---

### 관악구 공동교육과정 접근성

**관악구 강점: 서울대 캠퍼스 위치**

관악구는 서울대학교 관악캠퍼스가 위치해 있어 **대학 연계 공동교육과정** 접근성이 서울 내에서도 유리한 편입니다.

| 연계 가능 기관 | 내용 |
|-------------|------|
| **서울대학교** | 고교-대학 연계 심화과목 프로그램 (서울대 캠퍼스 수업) |
| **관악구 거점학교** | 당곡고(관악구 최대 규모, 공유캠퍼스 주관교)가 거점 역할 |
| **서울시 온라인 공동교육과정** | 전국 어디서나 수강 가능, AI·정보 과목 다수 개설 |

### 공동교육과정 신청 방법

```
1단계: 학기 초 성보고 교육과정 담당 교사에게 문의
       → 관악구 거점학교 개설 과목 목록 수령

2단계: 서울시교육청 공동교육과정 포털 확인
       → jccu.eduinfo.kr (또는 당해 연도 포털 주소)
       → 온라인·거점·대학 연계 과목 검색

3단계: 학교장 승인 후 수강 신청
       → 담임 → 교육과정 담당 → 학교장 결재

4단계: 이수 완료 시 학생부 자동 기재
```
