# 위조방지기술

지폐의 진위를 가르는 기술. 은솔·주환 지폐 주제탐구의 핵심 개념 중 하나.
1차 출처: [[sources/화폐_제3전시실_위조방지홍보관]].

## 간편 식별 3원칙 (초등 눈높이)
- **만져보고**: 볼록인쇄의 오톨도톨한 감촉.
- **기울여보고**: 색변환잉크 색 변화, 홀로그램, 요판잠상('WON'/숫자).
- **비추어보세요**: 숨은그림(초상), 숨은은선·숨은막대, 앞뒷면맞춤(태극).

## 주요 위조방지 요소
| 요소 | 설명 |
|------|------|
| 홀로그램 | 보는 각도에 따라 지도·태극·액면 숫자·4괘가 번갈아 나타남 |
| 색변환잉크 | 기울기에 따라 색이 변함 (오만원 자홍↔녹, 만·오천원 황금→녹, 천원 녹→청) |
| 숨은그림 | 빛에 비추면 초상이 나타남 (오만원 신사임당·만원 세종·오천원 율곡·천원 퇴계) |
| 볼록인쇄 | 손으로 만지면 도드라지는 감촉 |
| 은선(부분노출/숨은) | 용지에 삽입된 띠. 기울이면 태극 이동, 비추면 미세 문자·숫자 |
| 돌출은화 | 빛에 비추지 않아도 액면 숫자가 보이는 숨은그림 |
| 요판잠상 | 비스듬히 보면 'WON'/숫자가 드러남 |
| 미세문자 | 확대경으로 보이는 'BANK OF KOREA' 등 |
| 앞뒷면맞춤 | 앞·뒤 무늬가 합쳐져 태극 완성 |
| 형광잉크·은사 | 자외선 조사 시 형광·짧은 실선 |

## 권종별 요소 수
오만원 15종 · 만원 10종 · 오천원 10종 · 천원 8종. (상세: [[sources/화폐_제3전시실_위조방지홍보관]])

## 위조방지가 왜 면섬유와 연결되나
- [[concepts/지폐-재료]] 면섬유는 은선·홀로그램 등을 **용지 제조 단계에서 삽입**하기 좋아 위조를 어렵게 만든다.

## 위조의 정의·처벌
- 위조: 진짜로 오인할 가짜 화폐 제조 → 사형/무기/5년 이상 징역.
- 변조: 진짜 화폐 변형해 가치 변경. 위조지폐 알고 사용: 2년 이하 징역 또는 500만원 이하 벌금.
- 역사: 물품화폐 시기엔 위조가 거의 없었고 **지폐 발행 후 급증** → 기술 발전 동력. ([[concepts/물품화폐]])
