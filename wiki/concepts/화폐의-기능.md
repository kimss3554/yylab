# 화폐의 기능

화폐가 경제에서 수행하는 역할. 출처: [[sources/화폐_한중일고대화폐_체험학습지]], [[sources/화폐_경제박사_체험학습지]].

## 5가지 기능 (한·중·일 고대화폐展 기준)
| 기능 | 설명 |
|------|------|
| **가치척도** | 물건의 가치를 측정하는 단위 |
| **교환매개** | 물건을 사고팔 때 쓰는 교환 수단 |
| **가치저장** | 가치를 보관해 두는 수단 |
| **저축** | 미래를 위해 가치를 모아두는 기능 |
| **지급결제** | 거래의 대가를 치르는 수단 |

> 「경제박사」 체험학습지는 핵심 3기능(교환의 매개·가치의 척도·가치의 저장)으로 압축해 설명한다. 두 자료가 기능 분류 수에서 다르나 모순은 아니며, 저축·지급결제는 가치저장·교환매개의 확장으로 볼 수 있다.

## 화폐 진화와 기능
물품화폐 → 금속화폐 → 지폐로 발전하면서 운반·거래의 편의가 높아져 교환매개·지급결제 기능이 강화되었다. ([[concepts/물품화폐]], [[concepts/화폐의역사-한국]])

## 관련 경제 개념
- 물가 상승(인플레이션)·하락(디플레이션)은 화폐의 가치척도·가치저장 기능과 직결.
- 이자·세금·저축 등 경제활동의 기반. ([[concepts/한국은행]]이 통화신용정책으로 조절)
